Bases ortogonais e ortonormais

9.3 Bases ortogonais e ortonormais

Vimos que uma das importâncias do produto escalar é estender o conceito de ortogonalidade para espaços vetoriais mais gerais. A ortogonalidade traz consigo a noção de projeção ortogonal. Agora vamos ver algumas vantagens em se considerar bases de espaços vetoriais que são também conjuntos ortogonais; estas são conhecidas como bases ortogonais.

Exemplo 9.3.1.

A base canônica de ℝn\mathbb{R}^{n} forma um conjunto ortogonal e é, portanto, uma base ortogonal. Assim como no Exemplo 9.2.4, é possível representar qualquer vetor v→\vec{v} como

v→=(v→⋅e→1)⁢e→1+(v→⋅e→2)⁢e→2+⋯+(v→⋅e→n)⁢e→n.\vec{v}=\big{(}\vec{v}\cdot\vec{e}_{1}\big{)}\vec{e}_{1}+\big{(}\vec{v}\cdot% \vec{e}_{2}\big{)}\vec{e}_{2}+\cdots+\big{(}\vec{v}\cdot\vec{e}_{n}\big{)}\vec% {e}_{n}. (9.42)

Vejamos que esta propriedade da base canônica é, na realidade, uma propriedade de todas as bases ortogonais:

Teorema 9.3.1.

Seja ℬ={v→1,v→2,⋯,v→n}\mathcal{B}=\{\vec{v}_{1},\vec{v}_{2},\cdots,\vec{v}_{n}\} uma base ortogonal de ℝn\mathbb{R}^{n}. Então qualquer vetor v→∈ℝn\vec{v}\in\mathbb{R}^{n} pode ser representado como

v→=v→⋅v→1v→1⋅v→1⁢v→1+v→⋅v→2v→2⋅v→2⁢v→2+⋯+v→⋅v→nv→n⋅v→n⁢v→n.\vec{v}=\frac{\vec{v}\cdot\vec{v}_{1}}{\vec{v}_{1}\cdot\vec{v}_{1}}\,\vec{v}_{% 1}+\frac{\vec{v}\cdot\vec{v}_{2}}{\vec{v}_{2}\cdot\vec{v}_{2}}\,\vec{v}_{2}+% \cdots+\frac{\vec{v}\cdot\vec{v}_{n}}{\vec{v}_{n}\cdot\vec{v}_{n}}\,\vec{v}_{n}. (9.43)

Podemos escrever, de outra maneira, que

v→=projv→1⁡v→+projv→2⁡v→+⋯+projv→n⁡v→.\vec{v}=\operatorname{proj}_{\vec{v}_{1}}\vec{v}+\operatorname{proj}_{\vec{v}_% {2}}\vec{v}+\cdots+\operatorname{proj}_{\vec{v}_{n}}\vec{v}. (9.44)

Este resultado mostra que bases ortogonais são especiais e boas de trabalhar. Lembra que anteriormente, para encontrar os coeficientes de um vetor em uma base, tínhamos que encontrar c1,c2,⋯,cnc_{1},c_{2},\cdots,c_{n} resolvendo o sistema linear cuja forma vetorial é

c1⁢v→1+c2⁢v→2+⋯+cn⁢v→n=v→.c_{1}\vec{v}_{1}+c_{2}\vec{v}_{2}+\cdots+c_{n}\vec{v}_{n}=\vec{v}. (9.45)

O teorema acima afirma, em outras palavras, quando a base é ortogonal, os coeficientes do vetor v→\vec{v} na base ℬ\mathcal{B} são os números

cj=v→⋅v→jv→j⋅v→j.c_{j}=\frac{\vec{v}\cdot\vec{v}_{j}}{\vec{v}_{j}\cdot\vec{v}_{j}}. (9.46)
Justificativa do Teorema 9.3.1.

Se a representação do vetor v→\vec{v} na base ℬ\mathcal{B} for

v→=c1⁢v→1+c2⁢v→2+⋯+cn⁢v→n,\vec{v}=c_{1}\vec{v}_{1}+c_{2}\vec{v}_{2}+\cdots+c_{n}\vec{v}_{n}, (9.47)

então, fazendo o produto escalar desta expressão com v→j\vec{v}_{j} e utilizando as propriedades de ortogonalidade, concluímos que

v→⋅v→j=(c1⁢v→1+c2⁢v→2+⋯+cn⁢v→n)⋅v→j=c1⁢(v→1⋅v→j)+c2⁢(v→2⋅v→j)+⋯+cn⁢(v→n⋅v→j)=cj⁢(v→j⋅v→j),\vec{v}\cdot\vec{v}_{j}=\big{(}c_{1}\vec{v}_{1}+c_{2}\vec{v}_{2}+\cdots+c_{n}% \vec{v}_{n}\big{)}\cdot\vec{v}_{j}=c_{1}(\vec{v}_{1}\cdot\vec{v}_{j})+c_{2}(% \vec{v}_{2}\cdot\vec{v}_{j})+\cdots+c_{n}(\vec{v}_{n}\cdot\vec{v}_{j})=c_{j}% \big{(}\vec{v}_{j}\cdot\vec{v}_{j}\big{)}, (9.48)

pois v→i⋅v→j=0\vec{v}_{i}\cdot\vec{v}_{j}=0 sempre que i≠ji\neq j. Daí, concluímos, como queríamos, que os coeficientes devem satisfazer

cj=v→⋅v→jv→j⋅v→j.∎c_{j}=\frac{\vec{v}\cdot\vec{v}_{j}}{\vec{v}_{j}\cdot\vec{v}_{j}}.\qed (9.49)
Exemplo 9.3.2.

Vamos justificar que os vetores

u→=[42] e v→=[−12]\vec{u}=\begin{bmatrix}4\\ 2\end{bmatrix}\ \ \text{ e }\ \ \vec{v}=\begin{bmatrix}-1\\ 2\end{bmatrix} (9.50)

formam uma base ortogonal para ℝ2\mathbb{R}^{2}.

Em primeiro lugar, formam uma base, pois são vetores linearmente independentes e são dois (que é a dimensão de ℝ2\mathbb{R}^{2}). Agora, calculamos

u→⋅v→=4⋅(−1)+2⋅2=0.\vec{u}\cdot\vec{v}=4\cdot(-1)+2\cdot 2=0. (9.51)

Segue que u→\vec{u} e v→\vec{v} são também ortogonais e, logo, formam uma base ortogonal de ℝ2\mathbb{R}^{2}.

Para escrever um vetor x→=[2−7]\vec{x}=\begin{bmatrix}2\\ -7\end{bmatrix} nesta base, podemos calcular as projeções nos elementos da base (isto apenas vale se os elementos da base são ortogonais!):

x→=x→⋅u→u→⋅u→⁢u→+x→⋅v→v→⋅v→⁢v→=−620⁢u→+−165⁢v→=−0.3⁢u→−3.2⁢v→.⊲\vec{x}=\frac{\vec{x}\cdot\vec{u}}{\vec{u}\cdot\vec{u}}\,\vec{u}+\frac{\vec{x}% \cdot\vec{v}}{\vec{v}\cdot\vec{v}}\,\vec{v}=\frac{-6}{20}\,\vec{u}+\frac{-16}{% 5}\,\vec{v}=-0.3\,\vec{u}-3.2\,\vec{v}.\ \lhd (9.52)

Observe no exemplo acima que é muito mais fácil obter as componentes por ortogonalidade do que resolver sistemas lineares. Isto mesmo em dimensão baixa! Imagine se fosse um sistema 5×55\times 5 ou ainda maior.

Note que a base canônica, como no Exemplo 9.3.1, tem a propriedade adicional que ∥e→j∥=1\left\lVert\vec{e}_{j}\right\rVert=1 para todo índice jj. Isto fez com que a representação de vetores do Teorema 9.3.1 assumisse um formato ainda mais simples. Estas bases ortogonais que tem por elementos vetores unitários são conhecidas como bases ortonormais. Mais explicitamente, uma base ortonormal é uma base {v→1,v→2,⋯,v→n}\{\vec{v}_{1},\vec{v}_{2},\cdots,\vec{v}_{n}\} de um espaço vetorial que adicionalmente satisfaz

v→i⋅v→j=0 para i≠j e também ∥v→j∥=1 para todo j(⇔v→j⋅v→j=1)\vec{v}_{i}\cdot\vec{v}_{j}=0\text{ para }i\neq j\text{ e tamb\'{e}m }\left% \lVert\vec{v}_{j}\right\rVert=1\text{ para todo }j\ \ \big{(}\iff\vec{v}_{j}% \cdot\vec{v}_{j}=1\big{)} (9.53)

Uma consequência imediata é que, sendo {v→1,v→2,⋯,v→n}\{\vec{v}_{1},\vec{v}_{2},\cdots,\vec{v}_{n}\} uma base ortonormal de ℝn\mathbb{R}^{n}, podemos escrever

v→=(v→⋅v→1)⁢v→1+(v→⋅v→2)⁢v→2+⋯+(v→⋅v→n)⁢v→n.\vec{v}=\big{(}\vec{v}\cdot\vec{v}_{1}\big{)}\vec{v}_{1}+\big{(}\vec{v}\cdot% \vec{v}_{2}\big{)}\vec{v}_{2}+\cdots+\big{(}\vec{v}\cdot\vec{v}_{n}\big{)}\vec% {v}_{n}. (9.54)
Exemplo 9.3.3.

Os vetores

u→=[100],v→=[02/22/2],e ⁢w→=[02/2−2/2]\vec{u}=\begin{bmatrix}1\\ 0\\ 0\end{bmatrix},\ \vec{v}=\begin{bmatrix}0\\ \sqrt{2}/2\\ \sqrt{2}/2\end{bmatrix},\ \text{e }\vec{w}=\begin{bmatrix}0\\ \sqrt{2}/2\\ -\sqrt{2}/2\end{bmatrix} (9.55)

formam uma base ortonormal de ℝ3\mathbb{R}^{3}: verifique! Qualquer vetor x→∈ℝ3\vec{x}\in\mathbb{R}^{3} pode ser escrito como

x→=(x→⋅u→)⁢u→+(x→⋅v→)⁢v→+(x→⋅w→)⁢w→\vec{x}=\big{(}\vec{x}\cdot\vec{u}\big{)}\vec{u}+\big{(}\vec{x}\cdot\vec{v}% \big{)}\vec{v}+\big{(}\vec{x}\cdot\vec{w}\big{)}\vec{w} (9.56)

Por exemplo,

x→=[2−11]=2⁢u→+0⁢v→−2⁢w→=2⁢[100]+2⁢[02/2−2/2].\vec{x}=\begin{bmatrix}2\\ -1\\ 1\end{bmatrix}=2\vec{u}+0\vec{v}-\sqrt{2}\vec{w}=2\begin{bmatrix}1\\ 0\\ 0\end{bmatrix}+\sqrt{2}\begin{bmatrix}0\\ \sqrt{2}/2\\ -\sqrt{2}/2\end{bmatrix}. (9.57)

Observamos finalmente que bases ortogonais podem ser facilmente transformadas em bases ortonormais pelo processo de normalização.

Exemplo 9.3.4.

Os vetores

u→=[111],v→=[10−1]⁢e ⁢w→=[1−21]\vec{u}=\begin{bmatrix}1\\ 1\\ 1\end{bmatrix},\ \vec{v}=\begin{bmatrix}1\\ 0\\ -1\end{bmatrix}\ \text{e }\vec{w}=\begin{bmatrix}1\\ -2\\ 1\end{bmatrix} (9.58)

formam uma base ortogonal de ℝ3\mathbb{R}^{3}. Normalizando cada um dos vetores, podemos obter uma base ortonormal de ℝ3\mathbb{R}^{3}:

{∥u→∥=1+1+1=3∥v→∥=1+1=2∥w→∥=1+4+1=6\left\{\begin{array}[]{ll}\left\lVert\vec{u}\right\rVert=\sqrt{1+1+1}=\sqrt{3}% \\ \left\lVert\vec{v}\right\rVert=\sqrt{1+1}=\sqrt{2}\\ \left\lVert\vec{w}\right\rVert=\sqrt{1+4+1}=\sqrt{6}\\ \end{array}\right. (9.59)

Isto implica que os vetores

u→∥u→∥=13⁢[111]=[1/31/31/3],v→∥v→∥=12⁢[10−1]=[1/20−1/2]⁢e ⁢w→∥w→∥=16⁢[1−21]=[1/6−2/61/6]\frac{\vec{u}}{\left\lVert\vec{u}\right\rVert}=\frac{1}{\sqrt{3}}\begin{% bmatrix}1\\ 1\\ 1\end{bmatrix}=\begin{bmatrix}1/\sqrt{3}\\ 1/\sqrt{3}\\ 1/\sqrt{3}\end{bmatrix},\ \frac{\vec{v}}{\left\lVert\vec{v}\right\rVert}=\frac% {1}{\sqrt{2}}\begin{bmatrix}1\\ 0\\ -1\end{bmatrix}=\begin{bmatrix}1/\sqrt{2}\\ 0\\ -1/\sqrt{2}\end{bmatrix}\ \text{e }\frac{\vec{w}}{\left\lVert\vec{w}\right% \rVert}=\frac{1}{\sqrt{6}}\begin{bmatrix}1\\ -2\\ 1\end{bmatrix}=\begin{bmatrix}1/\sqrt{6}\\ -2/\sqrt{6}\\ 1/\sqrt{6}\end{bmatrix} (9.60)

formam um base ortonormal de ℝ3.⊲\mathbb{R}^{3}.\ \lhd

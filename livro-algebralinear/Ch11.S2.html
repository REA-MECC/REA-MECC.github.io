Regressão Linear Simples

11.2 Regressão Linear Simples

Vamos apresentar uma aplicação de algumas das técnicas do método de mínimos quadrados à Estatística ou Econometria. Uma regressão linear simples é uma equação, tipicamente da forma

y=a+b⁢x,y=a+bx, (11.21)

para estimar os valores (x,y)(x,y) apenas conhecendo alguns valores específicos (x1,y1),(x2,y2),…,(x_{1},y_{1}),(x_{2},y_{2}),\dots, (xk,yk)(x_{k},y_{k}). A ideia é tentar capturar como que mudanças na variável independente xx afetam a variável dependente yy (neste caso, supondo que esta dependência é linear).

Exemplo 11.2.1.
11 1 Exemplo adaptado de https://onlinecourses.science.psu.edu/stat501/node/257

Com o intuito de analisar se é razoável supor que há um relação linear entre a idade de um motorista e quão longe ele consegue ver, uma empresa (Last Resource, Inc., Bellefonte, PA) coletou dados de 30 motoristas. Para simplificar as nossas contas, vamos listar abaixo apenas alguns destes dados.

Idade Distância (em mm)
20 590
32 410
41 460
49 380
66 350

Podemos pensar em yy como a distância e em xx como a idade. Gostaríamos de achar uma relação linear da forma

y=a+b⁢x.y=a+bx. (11.22)

Desta forma, os dados obtidos implicam que

{a+20⁢b=590a+32⁢b=410a+41⁢b=460a+49⁢b=380a+66⁢b=350↭[120132141149166]⁢[ab]=[590410460380350]\left\{\begin{array}[]{ll}a+20b=590\\ a+32b=410\\ a+41b=460\\ a+49b=380\\ a+66b=350\end{array}\right.\ \ \leftrightsquigarrow\ \ \ \begin{bmatrix}1&20\\ 1&32\\ 1&41\\ 1&49\\ 1&66\\ \end{bmatrix}\begin{bmatrix}a\\ b\end{bmatrix}=\begin{bmatrix}590\\ 410\\ 460\\ 380\\ 350\end{bmatrix} (11.23)

Ora, dificilmente um sistema linear com duas incógnitas e cinco equações terá solução (só terá solução se todos os pontos do conjunto de dados estiverem perfeitamente alinhados em uma reta!). Consequentemente, vamos procurar por uma solução de mínimos quadrados. Isto é regressão linear simpes.

Denotando

A=[120132141149166]eb→=[590410460380350]A=\begin{bmatrix}1&20\\ 1&32\\ 1&41\\ 1&49\\ 1&66\\ \end{bmatrix}\quad\text{e}\quad\vec{b}=\begin{bmatrix}590\\ 410\\ 460\\ 380\\ 350\end{bmatrix} (11.24)

precisamos calcular

AT⁢A=[111112032414966]⁢[120132141149166]=[52082089832]A^{T}A=\begin{bmatrix}1&1&1&1&1\\ 20&32&41&49&66\end{bmatrix}\begin{bmatrix}1&20\\ 1&32\\ 1&41\\ 1&49\\ 1&66\\ \end{bmatrix}=\begin{bmatrix}5&208\\ 208&9832\\ \end{bmatrix} (11.25)

e

AT⁢b→=[111112032414966]⁢[590410460380350]=[219085500].A^{T}\vec{b}=\begin{bmatrix}1&1&1&1&1\\ 20&32&41&49&66\end{bmatrix}\begin{bmatrix}590\\ 410\\ 460\\ 380\\ 350\end{bmatrix}=\begin{bmatrix}2190\\ 85500\end{bmatrix}. (11.26)

Em seguida, a matriz associada aumentada pode ser reduzida por escalonamento:

[AT⁢A|b→]=[52082190208983285500]∼[10468510/73701−7005/1474]⟹{a≃635.7b≃−4.75.[\,A^{T}A\ |\ \vec{b}\,]=\begin{bmatrix}5&208&2190\\ 208&9832&85500\\ \end{bmatrix}\sim\begin{bmatrix}1&0&468510/737\\ 0&1&-7005/1474\\ \end{bmatrix}\implies\left\{\begin{array}[]{ll}a\simeq 635.7\\ b\simeq-4.75\end{array}\right.. (11.27)

Os números são feios, mas as contas feitas foram as que sempre fizemos ao escalonar uma matriz até sua forma escalonada reduzida.

A conclusão é que a reta de mínimos quadrados que melhor aproxima os nossos dados é a reta

y=a+b⁢x=635.7−4.75⁢x.y=a+bx=635.7-4.75x. (11.28)

O erro de mínimos quadrados nesta aproximação (ou norma do resíduo) pode ser calculado como

A⁢x→=[120132141149166]⁢[635.74.75]≃[730.7787.7830.45868.45949.2]≃projCol⁡A⁡b→⟹ erro =‖b→−A⁢x→‖≃947.3A\vec{x}=\begin{bmatrix}1&20\\ 1&32\\ 1&41\\ 1&49\\ 1&66\\ \end{bmatrix}\begin{bmatrix}635.7\\ 4.75\\ \end{bmatrix}\simeq\begin{bmatrix}730.7\\ 787.7\\ 830.45\\ 868.45\\ 949.2\\ \end{bmatrix}\simeq\operatorname{proj}_{\operatorname{Col}A}\vec{b}\implies% \text{ erro }=\|\vec{b}-A\vec{x}\|\simeq 947.3 (11.29)

Na figura, mostramos um esboço da reta que melhor aproxima os dados deste exemplo.⊲.\ \lhd

De uma maneira geral, para encontrar uma reta de melhor ajuste a uma quantidade de pontos (dados coletados de algum problema)

(x1,y1),(x2,y2),…,(xk,yk),(x_{1},y_{1}),\ (x_{2},y_{2}),\ \dots,(x_{k},y_{k}), (11.30)

devemos procurar pela solução (a,b)(a,b) de mínimos quadrados do sistema linear

{a+b⁢x1=y1a+b⁢x2=y2⋮a+b⁢xk=yk↭[1x11x2⋮⋮1xk]⁢[ab]=[y1y2⋮yk].\left\{\begin{array}[]{rl}a+bx_{1}&\!\!\!\!\!=y_{1}\\ a+bx_{2}&\!\!\!\!\!=y_{2}\\ \vdots&\\ a+bx_{k}&\!\!\!\!\!=y_{k}\\ \end{array}\right.\quad\leftrightsquigarrow\quad\begin{bmatrix}1&x_{1}\\ 1&x_{2}\\ \vdots&\vdots\\ 1&x_{k}\\ \end{bmatrix}\begin{bmatrix}a\\ b\end{bmatrix}=\begin{bmatrix}y_{1}\\ y_{2}\\ \vdots\\ y_{k}\end{bmatrix}. (11.31)

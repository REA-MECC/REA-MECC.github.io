Matrizes simétricas

12.1 Matrizes simétricas

Uma matriz simétrica é uma matriz que é igual à sua transposta. Para que esta definição faça sentido, apenas podemos considerar matrizes que são quadradas. De forma mais precisa, se A=[ai⁢j]A=\begin{bmatrix}a_{ij}\end{bmatrix} é uma matriz de ordem n×nn\times n, nós dizemos que AA é simétrica quando A=ATA=A^{T}.

Formas equivalentes de dizer que AA é simétrica incluem:

  1. (1)(1)
    ​

    A=ATA=A^{T};

  2. (2)(2)
    ​

    para todos os índices i,j∈{1,2,…,n}i,j\in\{1,2,\dots,n\}, temos que ai⁢j=aj⁢ia_{ij}=a_{ji};

  3. (3)(3)
    ​

    as colunas de AA são iguais às linhas de ATA^{T};

  4. (4)(4)
    ​

    as linhas de AA são iguais às colunas de ATA^{T}.

Exemplo 12.1.1.

As matrizes

[1−3−32],[1−3−31],[1−30−3−14046]⁢✓\begin{bmatrix}1&-3\\ -3&2\end{bmatrix},\ \ \begin{bmatrix}1&-3\\ -3&1\end{bmatrix},\ \ \begin{bmatrix}1&-3&0\\ -3&-1&4\\ 0&4&6\end{bmatrix}\checkmark (12.1)

são simétricas, enquanto as matrizes

[1−331],[13−32],[1−304−1−3041]⁢x\begin{bmatrix}1&-3\\ 3&1\end{bmatrix},\ \ \begin{bmatrix}1&3\\ -3&2\end{bmatrix},\ \ \begin{bmatrix}1&-3&0\\ 4&-1&-3\\ 0&4&1\end{bmatrix}\textsc{x} (12.2)

não são. Deve-se tomar o cuidado de verificar a simetria com relação à diagonal principal. Qualquer outro tipo de simetria não deve interferir na nossa análise.⊲.\ \lhd

Um dos resultados mais importantes de Álgebra Linear é o Teorema Espectral, que diz que toda matriz simétrica é diagonalizável por uma base de autovetores que é ortonormal. De forma mais precisa:

Teorema 12.1.1.

Suponhamos que AA é uma matriz simétrica com coeficientes reais. Então:

  1. (i)(i)
    ​

    todos os autovalores de AA são reais;

  2. (i⁢i)(ii)
    ​

    autovetores associados a autovalores distintos são ortogonais.

  3. (i⁢i⁢i)(iii)
    ​

    a dimensão do autoespaço associado a um autovalor é igual à multiplicidade deste autovalor como raiz do polinômio característico;

  4. (i⁢v)(iv)
    ​

    existe uma base ortonormal de ℝn\mathbb{R}^{n} formada por autovetores de AA;

  5. (v)(v)
    ​

    AA é diagonalizável por uma matriz de mudança de coordenadas ortonormal PP: a matriz PP cujas colunas são os autovetores (ortonormais) de AA é tal que

    PT⁢A⁢P=P−1⁢A⁢P=D.P^{T}AP=P^{-1}AP=D. (12.3)

Nós vamos apresentar uma prova deste resultado no final destas notas. Antes de mais nada, faremos alguns exemplos e aplicações que justificam a importância do Teorema Espectral. No entanto, é fortemente aconselhado que o leitor acompanhe os elementos da demonstração, para efetivamente entender as propriedades essenciais das matrizes simétricas.

12.1.1 Formas quadráticas

Nesta seção, apresentamos uma aplicação matemática de matrizes simétricas. Uma forma quadrática em ℝn\mathbb{R}^{n} é uma função polinomial em que todos os termos são de grau 2.

Exemplo 12.1.4.

A função Q:ℝ2→ℝQ:\mathbb{R}^{2}\to\mathbb{R}, definida por

Q⁢(x1,x2)=3⁢x12+x1⁢x2Q(x_{1},x_{2})=3x_{1}^{2}+x_{1}x_{2} (12.24)

é uma forma quadrática em ℝ2\mathbb{R}^{2}, já que cada um dos termos tem grau 2.

Mais geralmente, qualquer forma quadrática em ℝ2\mathbb{R}^{2} pode ser escrita como

Q⁢(x1,x2)=a⁢x12+2⁢b⁢x1⁢x2+c⁢x22,Q(x_{1},x_{2})=ax_{1}^{2}+2bx_{1}x_{2}+cx_{2}^{2}, (12.25)

onde a,b,c∈ℝa,b,c\in\mathbb{R}, já que os três termos acima são todos os possíveis monômios de grau dois em duas variáveis x1x_{1} e x2x_{2}. O coeficiente 22 que aparece no segundo termo é motivado pela fórmula mais geral (12.28) abaixo (observe que cada termo xi⁢xjx_{i}x_{j} aparece duas vezes na soma).

Exemplo 12.1.5.

A função Q:ℝ3→ℝQ:\mathbb{R}^{3}\to\mathbb{R}, definida por

Q⁢(x1,x2,x3)=x22−x32+3⁢x1⁢x2−8⁢x1⁢x3Q(x_{1},x_{2},x_{3})=x_{2}^{2}-x_{3}^{2}+3x_{1}x_{2}-8x_{1}x_{3} (12.26)

é uma forma quadrática em ℝ3\mathbb{R}^{3}. Qualquer forma quadrática em ℝ3\mathbb{R}^{3} pode ser escrita como

Q⁢(x1,x2)=a⁢x12+b⁢x22+c⁢x32+2⁢d⁢x1⁢x2+2⁢e⁢x1⁢x3+2⁢f⁢x2⁢x3,Q(x_{1},x_{2})=ax_{1}^{2}+bx_{2}^{2}+cx_{3}^{2}+2dx_{1}x_{2}+2ex_{1}x_{3}+2fx_% {2}x_{3}, (12.27)

onde a,b,c,d,e,f∈ℝa,b,c,d,e,f\in\mathbb{R}.

De forma geral, uma forma quadrática Q:ℝn→ℝQ:\mathbb{R}^{n}\to\mathbb{R} pode ser escrita como

Q⁢(x1,x2,…,xn)=∑i=1n∑j=1nai⁢j⁢xi⁢xj,Q(x_{1},x_{2},\dots,x_{n})=\sum_{i=1}^{n}\sum_{j=1}^{n}a_{ij}x_{i}x_{j}, (12.28)

onde cada um dos coeficientes ai⁢ja_{ij} são números reais. Podemos também supor que a matriz A=(ai⁢j)A=(a_{ij}) é simétrica.22 2 Caso Q⁢(x1,x2,…,xn)=∑i=1n∑j=1nbi⁢j⁢xi⁢xjQ(x_{1},x_{2},\dots,x_{n})=\sum_{i=1}^{n}\sum_{j=1}^{n}b_{ij}x_{i}x_{j} com uma matriz B=(bi⁢j)B=(b_{ij}) não simétrica, podemos escrever Q⁢(x1,x2,…,xn)=∑i=1nbi⁢i⁢xi2+∑i<j(bi⁢j+bj⁢i)⁢xi⁢xjQ(x_{1},x_{2},\dots,x_{n})=\sum_{i=1}^{n}b_{ii}x_{i}^{2}+\sum_{i<j}(b_{ij}+b_{% ji})x_{i}x_{j}. Definindo ai⁢j=bi⁢j+bj⁢i2a_{ij}=\frac{b_{ij}+b_{ji}}{2} temos também Q⁢(x1,x2,…,xn)=∑i=1nai⁢i⁢xi2+2⁢∑i<jai⁢j⁢xi⁢xjQ(x_{1},x_{2},\dots,x_{n})=\sum_{i=1}^{n}a_{ii}x_{i}^{2}+2\sum_{i<j}a_{ij}x_{i% }x_{j}. Esta forma é mais semelhante com as dos exemplos acima. Note como aparece o coeficiente 2 nos termos com i≠ji\neq j. Finalmente, observamos que A=(ai⁢j)A=(a_{ij}) é uma matriz simétrica e que ∑i=1n∑j=1nbi⁢j⁢xi⁢xj=∑i=1nai⁢i⁢xi2+2⁢∑i<jai⁢j⁢xi⁢xj=∑i=1n∑j=1nai⁢j⁢xi⁢xj.\sum_{i=1}^{n}\sum_{j=1}^{n}b_{ij}x_{i}x_{j}=\sum_{i=1}^{n}a_{ii}x_{i}^{2}+2% \sum_{i<j}a_{ij}x_{i}x_{j}=\sum_{i=1}^{n}\sum_{j=1}^{n}a_{ij}x_{i}x_{j}.

Notamos o seguinte: se AA é uma matriz de ordem m×nm\times n e x→∈ℝn\vec{x}\in\mathbb{R}^{n} é um vetor com nn componentes, então o vetor A⁢x→A\vec{x}, que é o produto de AA por x→\vec{x}, tem na sua ii-ésima componente o produto escalar entre a linha ii de AA e o vetor x→\vec{x}:

(A⁢x→)i=componente i de ⁢A⁢x→=∑j=1nai⁢j⁢xj.\left(A\vec{x}\right)_{i}=\text{componente $i$ de }A\vec{x}=\sum_{j=1}^{n}a_{% ij}x_{j}. (12.29)

Dessa forma, temos

Q⁢(x1,x2,…,xn)=∑i=1n∑j=1nai⁢j⁢xi⁢xj=∑i=1n(A⁢x→)i⁢xi=(A⁢x→)⋅x→.Q(x_{1},x_{2},\dots,x_{n})=\sum_{i=1}^{n}\sum_{j=1}^{n}a_{ij}x_{i}x_{j}=\sum_{% i=1}^{n}\left(A\vec{x}\right)_{i}x_{i}=\left(A\vec{x}\right)\cdot\vec{x}. (12.30)

E aí está a relação entre as formas quadráticas e matrizes simétricas. Acabamos de verificar que:

Qualquer forma quadrática em ℝn pode ser escrita na formaQ⁢(x1,x2,…,xn)=A⁢x→⋅x→,onde A é uma matriz simétrica de ordem n×n.\boxed{\begin{array}[]{c}\text{Qualquer forma quadr\'{a}tica em $\mathbb{R}^{n% }$ pode ser escrita na forma}\\ Q(x_{1},x_{2},\dots,x_{n})=A\vec{x}\cdot\vec{x},\\ \text{onde $A$ \'{e} uma matriz sim\'{e}trica de ordem $n\times n$.}\end{array}} (12.31)

Uma outra forma de escrever a mesma coisa (verifique a última igualdade como exercício) é

Q⁢(x1,x2,…,xn)=A⁢x→⋅x→=x→T⁢A⁢x→.Q(x_{1},x_{2},\dots,x_{n})=A\vec{x}\cdot\vec{x}=\vec{x}^{T}A\vec{x}. (12.32)

Q⁢(x1,x2)=3⁢x12+x1⁢x2=[x1x2]⁢[31/21/20]⁢[x1x2].Q(x_{1},x_{2})=3x_{1}^{2}+x_{1}x_{2}=\begin{bmatrix}x_{1}&x_{2}\end{bmatrix}% \begin{bmatrix}3&1/2\\ 1/2&0\end{bmatrix}\begin{bmatrix}x_{1}\\ x_{2}\end{bmatrix}. (12.34)

Os coeficientes de x12x_{1}^{2} e de x22x_{2}^{2} aparecem na diagonal principal. Antes de prosseguir, faremos uma conta para verificar que de fato tudo está de acordo:

[x1x2]⁢[31/21/20]⁢[x1x2]=[x1x2]⁢[3⁢x1+x2/2x1/2]=3⁢x12+x1⁢x22+x1⁢x22.\begin{bmatrix}x_{1}&x_{2}\end{bmatrix}\begin{bmatrix}3&1/2\\ 1/2&0\end{bmatrix}\begin{bmatrix}x_{1}\\ x_{2}\end{bmatrix}=\begin{bmatrix}x_{1}&x_{2}\end{bmatrix}\begin{bmatrix}3x_{1% }+x_{2}/2\\ x_{1}/2\end{bmatrix}=3x_{1}^{2}+\frac{x_{1}x_{2}}{2}+\frac{x_{1}x_{2}}{2}. (12.35)

12.1.2 Diagonalização de formas quadráticas

Embora não tenhamos dito explicitamente, estávamos trabalhando na base canônica de ℝn\mathbb{R}^{n}. Da forma usual, é possível fazer mudança de coordenadas. Do ponto de vista da matriz AA, que é simétrica e portanto diagonalizável, a base (ortonormal) de autovetores associados é a mais natural de todas. Suponhamos que

Q⁢(x→)=x→T⁢A⁢x→Q(\vec{x})=\vec{x}^{T}A\vec{x} (12.37)

e que ℬ={v→1,v→2,…,v→n}\mathcal{B}=\{\vec{v}_{1},\vec{v}_{2},\dots,\vec{v}_{n}\} é base ortonormal de ℝn\mathbb{R}^{n} formada por autovetores de AA. Desta forma,

P=[|||v→1v→2⋯v→n|||]⁢satisfaz⁢P−1⁢A⁢P=D,P=\begin{bmatrix}|&|&&|\\ \vec{v}_{1}&\vec{v}_{2}&\cdots&\vec{v}_{n}\\ |&|&&|\\ \end{bmatrix}\ \text{satisfaz}\ P^{-1}AP=D, (12.38)

onde DD é a matriz diagonal, com os respectivos autovalores na diagonal principal. Além disso, PP é ortogonal: PT=P−1P^{T}=P^{-1}. O vetor x→\vec{x} pode ser representado na base ℬ\mathcal{B} por:

x→=y1⁢v→1+y2⁢v→2+⋯+yn⁢v→n=P⁢y→,onde denotamosy→=[y1y2⋮yn]=[x→]ℬ.\vec{x}=y_{1}\vec{v}_{1}+y_{2}\vec{v}_{2}+\cdots+y_{n}\vec{v}_{n}=P\vec{y},% \quad\text{onde denotamos}\quad\vec{y}=\begin{bmatrix}y_{1}\\ y_{2}\\ \vdots\\ y_{n}\end{bmatrix}=\begin{bmatrix}\vec{x}\end{bmatrix}_{\mathcal{B}}. (12.39)

Agora, vamos escrever a forma quadrática na base ℬ\mathcal{B}, isto é, em termos do vetor y→\vec{y} (lembre que x→=P⁢y→⟹x→T=y→T⁢PT\vec{x}=P\vec{y}\implies\vec{x}^{T}=\vec{y}^{T}P^{T}):

x→T⁢A⁢x→=(y→T⁢PT)⁢A⁢(P⁢y→)=y→T⁢(PT⁢A⁢P)⁢y→=y→T⁢D⁢y→.\vec{x}^{T}A\vec{x}=(\vec{y}^{T}P^{T})A(P\vec{y})=\vec{y}^{T}(P^{T}AP)\vec{y}=% \vec{y}^{T}D\vec{y}. (12.40)

Isto significa que, ao mudar da base canônica (componentes xix_{i}) para a base ℬ\mathcal{B} (componentes yiy_{i}), temos

∑i,j=1nai⁢j⁢xi⁢xj=x→T⁢A⁢x→=y→T⁢D⁢y→=∑i=1nλi⁢yi2,\sum_{i,j=1}^{n}a_{ij}x_{i}x_{j}=\vec{x}^{T}A\vec{x}=\vec{y}^{T}D\vec{y}=\sum_% {i=1}^{n}\lambda_{i}y_{i}^{2}, (12.41)

onde os números λi∈ℝ\lambda_{i}\in\mathbb{R} são os autovalores da matriz AA (são reais porque a matriz é simétrica). Em outras palavras, a base (ortonormal) de autovetores de AA deixou transformou a nossa forma quadrática original em uma que não possui mais termos mistos, muito mais simples de trabalhar.

Exemplo 12.1.7.

Considere a forma quadrática

Q⁢(x1,x2)=6⁢x12−2⁢x1⁢x2+4⁢x22↭Q⁢(x→)=x→T⁢A⁢x→,ondeA=[6−2−23].Q(x_{1},x_{2})=6x_{1}^{2}-2x_{1}x_{2}+4x_{2}^{2}\quad\leftrightsquigarrow\quad Q% (\vec{x})=\vec{x}^{T}A\vec{x},\quad\text{onde}\quad A=\begin{bmatrix}6&-2\\ -2&3\end{bmatrix}. (12.42)

Vamos fazer uma mudança de coordenadas que transforma a forma quadrática QQ em uma forma sem termos mistos. Os autovalores de AA são as raízes de

p⁢(λ)=det(A−λ⁢I)=(6−λ)⁢(3−λ)−4=λ2−9⁢λ+14.p(\lambda)=\det(A-\lambda I)=(6-\lambda)(3-\lambda)-4=\lambda^{2}-9\lambda+14. (12.43)

Assim,

λ=9±81−562=9±52⟹λ1=2,λ2=7.\lambda=\frac{9\pm\sqrt{81-56}}{2}=\frac{9\pm 5}{2}\implies\lambda_{1}=2,% \lambda_{2}=7. (12.44)

Já sabemos que, ao obter uma base ortonormal de autovetores e escrevermos x→\vec{x} nesta base, a saber x→=P⁢y→\vec{x}=P\vec{y}, teremos

Q⁢(x→)=Q⁢(P⁢y→)=2⁢y12+7⁢y22.Q(\vec{x})=Q(P\vec{y})=2y_{1}^{2}+7y_{2}^{2}. (12.45)

Vamos verificar neste exemplo que isto de fato acontece.

  • •
    ​

    Autovetor unitário associado ao autovalor λ1=2\lambda_{1}=2:

    A−2⁢I=[4−2−21]∼[2−100]⟹v2=2⁢v1↝[12]↝v→1=[1/52/5].A-2I=\begin{bmatrix}4&-2\\ -2&1\end{bmatrix}\sim\begin{bmatrix}2&-1\\ 0&0\end{bmatrix}\implies v_{2}=2v_{1}\rightsquigarrow\begin{bmatrix}1\\ 2\end{bmatrix}\rightsquigarrow\vec{v}_{1}=\begin{bmatrix}1/\sqrt{5}\\ 2/\sqrt{5}\end{bmatrix}. (12.46)
  • •
    ​

    Autovetor unitário associado ao autovalor λ2=7\lambda_{2}=7:

    A−2⁢I=[−1−2−2−4]∼[−1−200]⟹v1=−2⁢v2↝[−21]↝v→2=[−2/51/5].A-2I=\begin{bmatrix}-1&-2\\ -2&-4\end{bmatrix}\sim\begin{bmatrix}-1&-2\\ 0&0\end{bmatrix}\implies v_{1}=-2v_{2}\rightsquigarrow\begin{bmatrix}-2\\ 1\end{bmatrix}\rightsquigarrow\vec{v}_{2}=\begin{bmatrix}-2/\sqrt{5}\\ 1/\sqrt{5}\end{bmatrix}. (12.47)

Portanto, encontramos uma base ortonormal (de modo que PP abaixo é matriz ortogonal e satisfaz P=PTP=P^{T}) e podemos escrever

P=[1/5−2/52/51/5]⟹PT⁢A⁢P=D=[2007]P=\begin{bmatrix}1/\sqrt{5}&-2/\sqrt{5}\\ 2/\sqrt{5}&1/\sqrt{5}\end{bmatrix}\implies P^{T}AP=D=\begin{bmatrix}2&0\\ 0&7\end{bmatrix} (12.48)

Assim,

Q⁢(x1,x2)=Q⁢(x→)=Q⁢(P⁢y→)=x→T⁢A⁢x→=y→T⁢A⁢y→=2⁢y12+7⁢y22.⊲Q(x_{1},x_{2})=Q(\vec{x})=Q(P\vec{y})=\vec{x}^{T}A\vec{x}=\vec{y}^{T}A\vec{y}=% 2y_{1}^{2}+7y_{2}^{2}.\ \lhd (12.49)

12.1.3 Aplicação à classificação de cônicas

Nesta subseção, nos restringimos a formas quadráticas em ℝ2\mathbb{R}^{2}. Uma curva de nível de uma forma quadrática, também conhecida como uma cônica, é um conjunto definido implicitamente pela equação

Q⁢(x1,x2)=k↭a⁢x12+b⁢x1⁢x2+c⁢x22=k,Q(x_{1},x_{2})=k\quad\leftrightsquigarrow\quad ax_{1}^{2}+bx_{1}x_{2}+cx_{2}^{% 2}=k, (12.50)

onde kk é uma constante. Em outras palavras, uma curva de nível é o conjunto de todos os pontos (x1,x2)(x_{1},x_{2}) que satisfazem Q⁢(x1,x2)=kQ(x_{1},x_{2})=k.

Já conhecemos alguns exemplos:

  • •
    ​

    x12+x22=r2x_{1}^{2}+x_{2}^{2}=r^{2} representa um círculo de raio rr;

  • •
    ​

    x12a2+x22b2=1\frac{x_{1}^{2}}{a^{2}}+\frac{x_{2}^{2}}{b^{2}}=1 representa uma elipse;

  • •
    ​

    x12a2−x22b2=1\frac{x_{1}^{2}}{a^{2}}-\frac{x_{2}^{2}}{b^{2}}=1 representa uma hipérbole.

Graficamente, vemos alguns exemplos na figura abaixo.

Vamos ver como que a diagonalização de formas simétricas se aplica para conseguirmos identificar cônicas de formas quadráticas que possuem termos mistos.

Vamos identificar a cônica cuja equação é

31⁢x12−10⁢3⁢x1⁢x2+21⁢x22=144.31x_{1}^{2}-10\sqrt{3}x_{1}x_{2}+21x_{2}^{2}=144. (12.51)

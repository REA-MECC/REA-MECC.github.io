Existência e unicidade de solução

1.4 Existência e unicidade de solução

1.4.1 Sistemas que possuem apenas uma solução

Os sistemas que vimos até agora possuíam apenas uma solução. Esta propriedade, como veremos, nem sempre é válida. No entanto, é fácil de identificar quando um sistema possui solução única analisando a forma escalonada da matriz associada: quando todas as colunas referentes às variáveis da equação possuírem posição de pivô. Por exemplo,

[1310−7−4],[121120−54−1100−3−3],[2000100100−120020600011].\left[\begin{array}[]{cc|c}1&3&1\\ 0&-7&-4\end{array}\right],\left[\begin{array}[]{ccc|c}1&2&1&12\\ 0&-5&4&-11\\ 0&0&-3&-3\end{array}\right],\left[\begin{array}[]{cccc|c}2&0&0&0&10\\ 0&1&0&0&-12\\ 0&0&2&0&6\\ 0&0&0&1&1\end{array}\right]. (1.45)

De fato, todas as matrizes com esta propriedade tem em sua forma escalonada reduzida a matriz identidade do lado esquerdo da barra que separa as variáveis e os valores do lado direito da igualdade do sistema:

[10−5/7014/7],[100501030011],[100050100−120010300011].\left[\begin{array}[]{cc|c}1&0&-5/7\\ 0&1&4/7\end{array}\right],\left[\begin{array}[]{ccc|c}1&0&0&5\\ 0&1&0&3\\ 0&0&1&1\end{array}\right],\left[\begin{array}[]{cccc|c}1&0&0&0&5\\ 0&1&0&0&-12\\ 0&0&1&0&3\\ 0&0&0&1&1\end{array}\right]. (1.46)

Escrevendo o sistema associado as estas últimas matrizes, é fácil de ver que a solução será única.

1.4.2 Sistemas impossíveis ou inconsistentes

Nem todo sistema linear possui solução. Um exemplo simples deste fato é

{x+y=1x+y=2.\left\{\begin{array}[]{ll}x+y=1\\ x+y=2\end{array}\right.. (1.47)

É claro que x+yx+y não pode ser ao mesmo tempo igual a 1 e igual a 2; portanto, o sistema não possui solução. O fato de o sistema não possuir solução – ser inconsistente – nem sempre é tão fácil de ser identificado apenas olhando para o sistema. Mas esta propriedade salta aos olhos quando analisamos a forma escalonada da matriz associada ao sistema. De fato, na matriz escalonada de um sistema inconsistente aparece uma linha da forma

[00000a],\left[\begin{array}[]{ccccc|c}0&0&0&0&0&a\end{array}\right], (1.48)

com uma constante a≠0a\neq 0. Esta linha representa no sistema uma equação do tipo 0=a≠00=a\neq 0, que é impossível de ser satisfeita.

No sistema da fórmula (1.47) acima, temos a matriz aumentada

[111112]→−ℓ1+ℓ2⁢ em ⁢ℓ2[111001].\left[\begin{array}[]{cc|c}1&1&1\\ 1&1&2\end{array}\right]\xrightarrow{-\ell_{1}+\ell_{2}\text{ em }\ell_{2}}% \quad\left[\begin{array}[]{cc|c}1&1&1\\ 0&0&1\end{array}\right]. (1.49)

Logo, nosso sistema é equivalente ao sistema impossível

{x+y=10=1.\left\{\begin{array}[]{rl}x+y&=1\\ 0&=1\end{array}\right.. (1.50)

1.4.3 Sistemas com mais de uma solução

Com exceção da subseção anterior, todos os sistemas que resolvemos anteriormente possuíam apenas uma solução. Vejamos um exemplo onde este não é o caso.

Exemplo 1.4.1.

Resolver

{3⁢x2−6⁢x3+6⁢x4+4⁢x5=−53⁢x1−7⁢x2+8⁢x3−5⁢x4+8⁢x5=93⁢x1−9⁢x2+12⁢x3−9⁢x4+6⁢x5=15.\left\{\begin{array}[]{rrrrrrl}&3x_{2}&-6x_{3}&+6x_{4}&+4x_{5}&=&-5\\ 3x_{1}&-7x_{2}&+8x_{3}&-5x_{4}&+8x_{5}&=&9\\ 3x_{1}&-9x_{2}&+12x_{3}&-9x_{4}&+6x_{5}&=&15\end{array}\right.. (1.51)

A matriz aumentada associada a este sistema é

[03−664−53−78−5893−912−9615].\left[\begin{array}[]{ccccc|c}0&3&-6&6&4&-5\\ 3&-7&8&-5&8&9\\ 3&-9&12&-9&6&15\end{array}\right]. (1.52)

Transformando esta matriz em sua forma escalonada reduzida (faça todos os cálculos!), obtemos

[10−230−2401−220−7000014],\left[\begin{array}[]{ccccc|c}1&0&-2&3&0&-24\\ 0&1&-2&2&0&-7\\ 0&0&0&0&1&4\end{array}\right], (1.53)

que é equivalente ao sistema linear

{x1−2⁢x3+3⁢x4=−24x2−2⁢x3+2⁢x4=−7x5=4\left\{\begin{array}[]{rcl}x_{1}-2x_{3}+3x_{4}&=&-24\\ x_{2}-2x_{3}+2x_{4}&=&-7\\ x_{5}&=&4\end{array}\right. (1.54)

Neste sistema x5x_{5} possui um valor fixo igual a 4, mas as outras variáveis tem dois graus de liberdade. De fato, atribuindo valores quaisquer para x3x_{3} e x4x_{4}, obtemos os valores de x1x_{1} e x2x_{2} pelas duas primeiras equações acima e temos uma solução do sistema. Diferentes escolhas dos parâmetros x3x_{3} e x4x_{4} geram soluções diferentes.

Uma forma sistemática de analisar a situação é a seguinte: As variáveis correspondentes a colunas que possuem posição de pivô são chamadas de variáveis dependentes e as demais são chamadas de variáveis livres. Assim, podemos escrever:

{x1=2⁢x3−3⁢x4−24x2=2⁢x3−2⁢x4−7x3,x4⁢ livres x5=4\left\{\begin{array}[]{l}x_{1}=2x_{3}-3x_{4}-24\\ x_{2}=2x_{3}-2x_{4}-7\\ x_{3},x_{4}\text{ livres }\\ x_{5}=4\end{array}\right. (1.55)

e o sistema possui infinitas soluções.⊲\lhd

1.4.4 Conclusões

Ao analisar a matriz aumentada associada a um sistema linear, concluiremos inevitavelmente que uma das seguintes situações é válida:

  • •
    ​

    Todas as colunas referentes às variáveis do sistema possuem posição de pivô, e.g,

    [13−660−811π0012−9.]\left[\begin{array}[]{ccc|c}1&3&-6&6\\ 0&-8&11&\pi\\ 0&0&12&-9.\end{array}\right] (1.56)

    Neste caso, vimos que o sistema possui apenas uma solução. Nota que esta possibilidade apenas é possível se tivermos um sistema com o número de equações igual ao número de variáveis.

  • •
    ​

    Caso a coluna referentes à alguma variável do sistema não possua posição de pivô, teremos duas possibilidades

    1. (i)(i)
      ​

      Alguma coluna não é coluna pivô mas não existem linhas inconsistentes, e.g,

      [132−6260−8411−1−3000121−9.]\left[\begin{array}[]{ccccc|c}1&3&2&-6&2&6\\ 0&-8&4&11&-1&-3\\ 0&0&0&12&1&-9.\end{array}\right] (1.57)

      Neste caso, as variáveis referentes a colunas que não são pivô (na matriz acima, colunas 3 e 5) podem ser consideradas variáveis livres e o sistema possui infinitas soluções.

    2. (i⁢i)(ii)
      ​

      Alguma coluna não é coluna pivô mas existem linhas que são inconsistentes, e.g,

      [132−6260−8411−1100000−9.]\left[\begin{array}[]{ccccc|c}1&3&2&-6&2&6\\ 0&-8&4&11&-1&1\\ 0&0&0&0&0&-9.\end{array}\right] (1.58)

      Neste caso, não existem soluções para o sistema, pois uma das equações é impossível de ser satisfeita.

Concluímos assim que um sistema linear ou não possui solução, ou tem apenas uma solução ou então possui infinitas soluções. São estes três todos os casos possíveis e podemos decidir em qual estamos apenas olhando para a forma escalonada da matriz aumentada associada ao sistema.

ER 1.4.1.

Faça o escalonamento da seguinte matriz e diga se o sistema possui infinitas soluções, uma solução ou se ele é impossível .

[1468−2−824−4−21001−32]\left[\begin{array}[]{ccc|c}1&4&6&8\\ -2&-8&2&4\\ -4&-2&1&0\\ 0&1&-3&2\end{array}\right] (1.59)
Solução.

Para começar, vamos

[1468−2−824−4−21001−32]→2⁢ℓ1+ℓ2⁢ em ⁢ℓ2[1468001420−4−21001−32].→4⁢ℓ1+ℓ3⁢ em ⁢ℓ3[1468001420014253201−32].\left[\begin{array}[]{ccc|c}1&4&6&8\\ -2&-8&2&4\\ -4&-2&1&0\\ 0&1&-3&2\end{array}\right]\quad\xrightarrow{2\ell_{1}+\ell_{2}\text{ em }\ell_% {2}}\quad\left[\begin{array}[]{ccc|c}1&4&6&8\\ 0&0&14&20\\ -4&-2&1&0\\ 0&1&-3&2\end{array}\right].\quad\xrightarrow{4\ell_{1}+\ell_{3}\text{ em }\ell% _{3}}\quad\left[\begin{array}[]{ccc|c}1&4&6&8\\ 0&0&14&20\\ 0&14&25&32\\ 0&1&-3&2\end{array}\right]. (1.60)

Percebemos então que não há mais nenhum elemento abaixo do primeiro elemento da primeira coluna, então trocamos de lugar a linha 2 com a linha 4 (assim facilitará os cálculos), continuamos com o resto das operações elementares necessárias e obtemos:

[146801−320142532001420]→−14⁢ℓ2+ℓ3⁢ em ⁢ℓ3[146801−3200674001420]→−1467⁢ℓ3+ℓ4⁢ em ⁢ℓ4[146801−3200674000128467]\left[\begin{array}[]{ccc|c}1&4&6&8\\ 0&1&-3&2\\ 0&14&25&32\\ 0&0&14&20\end{array}\right]\quad\xrightarrow{-14\ell_{2}+\ell_{3}\text{ em }% \ell_{3}}\quad\left[\begin{array}[]{ccc|c}1&4&6&8\\ 0&1&-3&2\\ 0&0&67&4\\ 0&0&14&20\end{array}\right]\quad\xrightarrow{\frac{-14}{67}\ell_{3}+\ell_{4}% \text{ em }\ell_{4}}\quad\left[\begin{array}[]{ccc|c}1&4&6&8\\ 0&1&-3&2\\ 0&0&67&4\\ 0&0&0&\frac{1284}{67}\end{array}\right] (1.61)

Observamos que na última linha há uma inconsistência do tipo 0=1284670=\frac{1284}{67} e, portanto, o sistema é impossível ⊲\lhd. ∎

ER 1.4.2.

Determine o(s) valor(es) de kk que fazem com que a matriz seja a matriz de um sistema linear com solução.

[122−3k−1].\left[\begin{array}[]{cc|c}1&2&2\\ -3&k&-1\end{array}\right]. (1.62)
Solução.

Fazendo uma operação elementar para simplificar, obtemos:

[122−3k−1]→3⁢ℓ1+ℓ2⁢ em ⁢ℓ2[1220k+65]\left[\begin{array}[]{cc|c}1&2&2\\ -3&k&-1\end{array}\right]\quad\xrightarrow{3\ell_{1}+\ell_{2}\text{ em }\ell_{% 2}}\quad\left[\begin{array}[]{cc|c}1&2&2\\ 0&k+6&5\end{array}\right] (1.63)

Assim, para que o sistema possua solução não podemos ter a inconsistência já mencionada, nesse caso de 5=05=0. Logo, temos que:

k+6≠0⟹k≠−6k+6\neq 0\implies k\neq-6 (1.64)

Isso significa que para qualquer valor de kk, exceto o −6-6, o sistema tem solução ⊲\lhd. ∎

Diagonalização

8.3 Diagonalização

Matrizes diagonais são matrizes da forma

[λ100⋯00λ20⋯000λ3⋯0⋮⋮⋮⋱⋮000⋯λn].\begin{bmatrix}\lambda_{1}&0&0&\cdots&0\\ 0&\lambda_{2}&0&\cdots&0\\ 0&0&\lambda_{3}&\cdots&0\\ \vdots&\vdots&\vdots&\ddots&\vdots\\ 0&0&0&\cdots&\lambda_{n}\\ \end{bmatrix}. (8.41)

Já que são triangulares, seus autovalores são os elementos da diagonal principal. Observamos também que os autovetores associados são os elementos da base canônica de ℝn\mathbb{R}^{n}.

Na realidade, o fato de se conseguir formar uma base com autovetores de uma matriz está intimamente relacionado com se obter uma matriz diagonal semelhante a AA. De fato, vamos justificar que, sempre que for possível formar uma base do espaço ℝn\mathbb{R}^{n} apenas com autovetores de uma matriz quadrada AA, então é possível fazer uma mudança de base de modo que a obter uma matriz diagonal, com os autovalores na diagonal principal. Este procedimento é chamado de diagonalização da matriz AA, por motivos óbvios. O método, que é surpreendentemente simples (mas, na prática, trabalhoso), consiste em montar uma matriz PP com os autovetores de AA e efetuar multiplicações de matrizes:

P=[|||v→1v→2⋯v→n|||]⟹P−1⁢A⁢P=D=[λ10⋯00λ2⋯0⋮⋮⋱⋮00⋯λn].P=\begin{bmatrix}|&|&&|\\ \vec{v}_{1}&\vec{v}_{2}&\cdots&\vec{v}_{n}\\ |&|&&|\\ \end{bmatrix}\implies P^{-1}AP=D=\begin{bmatrix}\lambda_{1}&0&\cdots&0\\ 0&\lambda_{2}&\cdots&0\\ \vdots&\vdots&\ddots&\vdots\\ 0&0&\cdots&\lambda_{n}\\ \end{bmatrix}. (8.42)
Justificativa do método de diagonalização.

Vamos verificar que P−1⁢A⁢P=DP^{-1}AP=D mostrando que as colunas da matriz P−1⁢A⁢PP^{-1}AP são iguais às colunas de DD. Pela forma que construímos a matriz PP, sabemos que a primeira coluna de PP é v→1\vec{v}_{1}:

P⁢e→1=v→1ou, ainda, e→1=P−1⁢v→1.P\vec{e}_{1}=\vec{v}_{1}\qquad\qquad\text{ou, ainda, }\qquad\vec{e}_{1}=P^{-1}% \vec{v}_{1}. (8.43)

Assim, já que A⁢v→1=λ1⁢v→1A\vec{v}_{1}=\lambda_{1}\vec{v}_{1}, temos

primeira coluna de ⁢P−1⁢A⁢P=P−1⁢A⁢P⁢e→1=P−1⁢A⁢v→1=P−1⁢λ1⁢v→1=λ1⁢P−1⁢v→1=λ1⁢e→1=primeira coluna de ⁢D.\begin{split}\text{primeira coluna de }P^{-1}AP&=P^{-1}AP\vec{e}_{1}\\ &=P^{-1}A\vec{v}_{1}\\ &=P^{-1}\lambda_{1}\vec{v}_{1}\\ &=\lambda_{1}P^{-1}\vec{v}_{1}\\ &=\lambda_{1}\vec{e}_{1}\\ &=\text{primeira coluna de }D.\end{split} (8.44)

Este mesmo raciocínio, repetido nn vezes, para cada uma das colunas, prova que todas as colunas são iguais; portanto, P−1⁢A⁢P=DP^{-1}AP=D. ∎

Exemplo 8.3.1 (de volta ao Exemplo 8.2.5).

Vimos que, para a matriz

A=[5816418−4−4−11],A=\left[\begin{array}[]{ccc}5&8&16\\ 4&1&8\\ -4&-4&-11\\ \end{array}\right], (8.45)

temos autovalores λ1=1\lambda_{1}=1 e λ2=−3\lambda_{2}=-3 e autoespaços associados:

Nul⁡(A−I)=Span⁡{[−2−11]};Nul⁡(A−I)=Span⁡{[−110],[−201]}.\operatorname{Nul}(A-I)=\operatorname{Span}\left\{\left[\begin{array}[]{c}-2\\ -1\\ 1\\ \end{array}\right]\right\};\qquad\operatorname{Nul}(A-I)=\operatorname{Span}% \left\{\left[\begin{array}[]{c}-1\\ 1\\ 0\\ \end{array}\right],\left[\begin{array}[]{ccc}-2\\ 0\\ 1\\ \end{array}\right]\right\}. (8.46)

Montamos a matriz

P=[−2−1−2−110101]P=\begin{bmatrix}-2&-1&-2\\ -1&1&0\\ 1&0&1\end{bmatrix} (8.47)

Por escalonamento, podemos calcular a matriz inversa

P−1=[−1−1−2−10−2113]P^{-1}=\begin{bmatrix}-1&-1&-2\\ -1&0&-2\\ 1&1&3\end{bmatrix} (8.48)

e temos

P−1⁢A⁢P\displaystyle P^{-1}AP =[−1−1−2−10−2113]⁢[5816418−4−4−11]⁢[−2−1−2−110101]\displaystyle=\begin{bmatrix}-1&-1&-2\\ -1&0&-2\\ 1&1&3\end{bmatrix}\begin{bmatrix}5&8&16\\ 4&1&8\\ -4&-4&-11\\ \end{bmatrix}\begin{bmatrix}-2&-1&-2\\ -1&1&0\\ 1&0&1\end{bmatrix}
=[−1−1−2306−3−3−9]⁢[−2−1−2−110101]=[1000−3000−3].⊲\displaystyle=\begin{bmatrix}-1&-1&-2\\ 3&0&6\\ -3&-3&-9\end{bmatrix}\begin{bmatrix}-2&-1&-2\\ -1&1&0\\ 1&0&1\end{bmatrix}=\begin{bmatrix}1&0&0\\ 0&-3&0\\ 0&0&-3\end{bmatrix}.\ \lhd

Embora o método de diagonalização seja bastante transparente, pois basta seguir uma sequência de passos, ele é muito trabalhoso! Não devemos nos iludir com a aparente simplicidade das contas no Exemplo 8.3.1, pois as contas já haviam sido feitas anteriormente (além de que “escondemos” o cálculo da inversa P−1P^{-1}). Na totalidade, os passos envolvem:

  • •
    ​

    Calcular o polinômio característico p⁢(λ)=det(A−λ⁢I)p(\lambda)=\det(A-\lambda I). Aqui já entra uma quantidade significativa de cálculos, cuja quantidade aumenta de acordo com a dimensão do problema;

  • •
    ​

    Encontrar as raízes do polinômio característico. Este passo pode ser muito complicado, já que muitas vezes não é possível encontrar raízes explicitamente;

  • •
    ​

    Depois de conhecidos os autovalores, calcular cada um dos autoespaços associados, o que demanda a resolução de tantos sistemas lineares quantos forem os autovalores distintos da matriz;

  • •
    ​

    Caso todos os geradores de todos os autoespaços formem uma base para o espaço ℝn\mathbb{R}^{n}, a matriz AA é diagonalizável e formamos a matriz PP como indicado acima;

  • •
    ​

    Observe que, se estivermos apenas interessados em encontrar uma base de autovetores e uma matriz diagonal equivalente a AA, então não há necessidade de se calcular P−1P^{-1}, pois já sabemos que o método funciona e que P−1⁢A⁢P=DP^{-1}AP=D. No entanto, uma descrição mais completa pode ser necessária nas principais aplicações, de modo que ainda seria preciso uma quantidade grande de cálculos para determinar a matriz P−1P^{-1}.

Nem todas as matrizes são diagonalizáveis, como acima. Já vimos que isto é possível quando uma base de autovetores existe. No exemplo

A=[2102]A=\begin{bmatrix}2&1\\ 0&2\end{bmatrix} (8.49)

não há dois autovetores linearmente independentes e, portanto, não é possível diagonalizar a matriz AA.

Em geral, valem as seguintes propriedades: seja AA uma matriz de ordem n×nn\times n. Então:

  • •
    ​

    Se AA possui nn autovalores reais distintos, então AA possui uma base de autovetores e é diagonalizável, pois possui um autovetor associado a cada um dos seus autovalores distintos. Isto é consequência de um resultado que diz que se temos kk autovetores associados a kk autovalores distintos, então este conjunto de kk autovetores é LI.

  • •
    ​

    No caso de AA possuir autovalores reais com multiplicidade maior do que 11, AA apenas será diagonalizável se cada autovalor de multiplicidade kk tiver kk autovetores linearmente independentes. Em outras palavras, só será possível formar uma base de autovetores de AA quando

    dim⁡Nul⁡(A−λ⁢I)= multiplicidade do autovalor ⁢λ.\operatorname{dim}\operatorname{Nul}(A-\lambda I)=\text{ multiplicidade do % autovalor }\lambda. (8.50)

    Na notação introduzida não seção anterior, isto é o mesmo que dizer que, se AA for diagonalizável, então as multiplicidades algébrica e geométrica são iguais.

  • •
    ​

    Caso dimNul⁡(A−λ⁢I)\dim\operatorname{Nul}(A-\lambda I) seja menor do que a multiplicidade do autovalor λ\lambda ou AA possua autovalores complexos, então AA não é diagonalizável.

Exemplo 8.3.2.

A matriz

A=[21−12]A=\begin{bmatrix}2&1\\ -1&2\end{bmatrix} (8.51)

tem polinômio característico

p⁢(λ)=det[2−λ1−12−λ]=(2−λ)2+1=λ2−4⁢λ+5p(\lambda)=\det\begin{bmatrix}2-\lambda&1\\ -1&2-\lambda\end{bmatrix}=(2-\lambda)^{2}+1=\lambda^{2}-4\lambda+5 (8.52)

cujas raízes são λ1=4+3⁢i2\lambda_{1}=\frac{4+3i}{2} e λ2=4−3⁢i2\lambda_{2}=\frac{4-3i}{2}. Logo, AA não é diagonalizável.

Dependência e independência linear

3.1 Dependência e independência linear

Nós vimos, por definição, que um vetor v→∈ℝm\vec{v}\in\mathbb{R}^{m} é combinação linear dos kk vetores v→1,v→2,…,v→k∈ℝm\vec{v}_{1},\vec{v}_{2},\dots,\vec{v}_{k}\in\mathbb{R}^{m} quando conseguirmos encontrar números reais x1,x2,…,xkx_{1},x_{2},\dots,x_{k} tais que

x1⁢v→1+x2⁢v→2+⋯+xk⁢v→k=v→.x_{1}\vec{v}_{1}+x_{2}\vec{v}_{2}+\cdots+x_{k}\vec{v}_{k}=\vec{v}. (3.1)

Vimos também que para decidir se um vetor é combinação linear de outros, devemos verificar se existem estes números reais x1,x2,…,xkx_{1},x_{2},\dots,x_{k}. Em coordenadas, isto é equivalente a resolver o sistema linear:

[v11v12v13⋯v1⁢kv21v22v23⋯v2⁢kv31v32v33⋯v3⁢k⋮⋮⋮⋮vm⁢1vm⁢2vm⁢3⋯vm⁢k]⁢[x1x2x3⋮xk]=[b1b2b3⋮bk].\left[\begin{array}[]{ccccc}v_{11}&v_{12}&v_{13}&\cdots&v_{1k}\\ v_{21}&v_{22}&v_{23}&\cdots&v_{2k}\\ v_{31}&v_{32}&v_{33}&\cdots&v_{3k}\\ \vdots&\vdots&\vdots&&\vdots\\ v_{m1}&v_{m2}&v_{m3}&\cdots&v_{mk}\end{array}\right]\left[\begin{array}[]{c}x_% {1}\\ x_{2}\\ x_{3}\\ \vdots\\ x_{k}\end{array}\right]=\left[\begin{array}[]{c}b_{1}\\ b_{2}\\ b_{3}\\ \vdots\\ b_{k}\end{array}\right]. (3.2)

Agora, dizemos que os vetores v→1,v→2,…,v→k∈ℝm\vec{v}_{1},\vec{v}_{2},\dots,\vec{v}_{k}\in\mathbb{R}^{m} são linearmente independentes (LI) se nenhum dos vetores puder ser escrito como combinação linear dos demais. Uma forma de escrever isto matematicamente é a seguinte:

Se x1v→1+x2v→2+⋯+xkv→k=0→, então x1=x2=⋯=xk=0.\boxed{\text{Se }x_{1}\vec{v}_{1}+x_{2}\vec{v}_{2}+\cdots+x_{k}\vec{v}_{k}=% \vec{0},\text{ ent\~{a}o }x_{1}=x_{2}=\cdots=x_{k}=0.} (3.3)

De fato, imaginem que pelo menos um dos coeficientes acima fosse diferente de zero, digamos x2≠0x_{2}\neq 0. Daí podemos dividir por x2x_{2} e conseguiríamos escrever o vetor v→2\vec{v}_{2} como combinação linear dos demais:

v→2=−x1x2⁢v→1−x3x2⁢v→3−⋯−xkx2⁢v→k.\vec{v}_{2}=-\frac{x_{1}}{x_{2}}\,\vec{v}_{1}-\frac{x_{3}}{x_{2}}\,\vec{v}_{3}% -\cdots-\frac{x_{k}}{x_{2}}\,\vec{v}_{k}. (3.4)

Se os vetores v→1,v→2,…,v→k∈ℝm\vec{v}_{1},\vec{v}_{2},\dots,\vec{v}_{k}\in\mathbb{R}^{m} não forem linearmente independentes, então nós dizemos que eles são linearmente dependentes (LD).

Exemplo 3.1.1.

Vamos verificar se os vetores

[100],[110],[111]\left[\begin{array}[]{c}1\\ 0\\ 0\end{array}\right],\quad\left[\begin{array}[]{c}1\\ 1\\ 0\end{array}\right],\quad\left[\begin{array}[]{c}1\\ 1\\ 1\end{array}\right] (3.5)

são LI ou LD.

Para verificar se são LI, nós devemos considerar a equação vetorial

x1⁢[100]+x2⁢[110]+x3⁢[111]=0→=[000].x_{1}\left[\begin{array}[]{c}1\\ 0\\ 0\end{array}\right]+x_{2}\left[\begin{array}[]{c}1\\ 1\\ 0\end{array}\right]+x_{3}\left[\begin{array}[]{c}1\\ 1\\ 1\end{array}\right]=\vec{0}=\left[\begin{array}[]{c}0\\ 0\\ 0\end{array}\right]. (3.6)

Como a equação é homogênea, temos pelo menos a solução trivial: x1=0x_{1}=0, x2=0x_{2}=0 e x3=0x_{3}=0. Se esta for a única solução, então os vetores são LI. Se existir alguma outra solução que não seja a trivial, então os vetores são LD.

Resolver a equação vetorial equivale a resolver o sistema linear

[111011001]⁢[x1x2x3]=[000].\left[\begin{array}[]{ccc}1&1&1\\ 0&1&1\\ 0&0&1\end{array}\right]\left[\begin{array}[]{c}x_{1}\\ x_{2}\\ x_{3}\end{array}\right]=\left[\begin{array}[]{c}0\\ 0\\ 0\end{array}\right]. (3.7)

É fácil de ver que a única solução deste sistema é a trivial e, portanto, os vetores são LI. Se este sistema não fosse fácil de resolver, deveríamos começar a resolvê-lo por escalonamento, como de costume. ⊲\ \lhd

Exemplo 3.1.2.

Analisamos agora os vetores

[101],[110],[111],[−121].\left[\begin{array}[]{c}1\\ 0\\ 1\end{array}\right],\quad\left[\begin{array}[]{c}1\\ 1\\ 0\end{array}\right],\quad\left[\begin{array}[]{c}1\\ 1\\ 1\end{array}\right],\quad\left[\begin{array}[]{c}-1\\ 2\\ 1\end{array}\right]. (3.8)

Como no exemplo anterior, para verificar se são LI, nós devemos considerar a equação vetorial

x1⁢[101]+x2⁢[110]+x3⁢[111]+x4⁢[−121]=[000].x_{1}\left[\begin{array}[]{c}1\\ 0\\ 1\end{array}\right]+x_{2}\left[\begin{array}[]{c}1\\ 1\\ 0\end{array}\right]+x_{3}\left[\begin{array}[]{c}1\\ 1\\ 1\end{array}\right]+x_{4}\left[\begin{array}[]{c}-1\\ 2\\ 1\end{array}\right]=\left[\begin{array}[]{c}0\\ 0\\ 0\end{array}\right]. (3.9)

Como a equação é homogênea, temos pelo menos a solução trivial: x1=0x_{1}=0, x2=0x_{2}=0 e x3=0x_{3}=0. Se esta for a única solução, então os vetores são LI. Se existir alguma outra solução que não seja a trivial, então os vetores são LD.

Resolver a equação vetorial equivale a resolver o sistema linear

[111−101121011]⁢[x1x2x3x4]=[000].\left[\begin{array}[]{cccc}1&1&1&-1\\ 0&1&1&2\\ 1&0&1&1\end{array}\right]\left[\begin{array}[]{c}x_{1}\\ x_{2}\\ x_{3}\\ x_{4}\end{array}\right]=\left[\begin{array}[]{c}0\\ 0\\ 0\end{array}\right]. (3.10)

Em um sistema com mais variáveis do que equações, podemos não ter soluções (no caso de alguma linha inconsistente) ou podemos ter infinitas soluções (no caso de haver variáveis livres). Não é possível ter unicidade. Como já sabemos que em um sistema homogêneo, sempre temos pelo menos a solução trivial x1=0x_{1}=0, x2=0x_{2}=0, x3=0x_{3}=0 e x4=0x_{4}=0, podemos concluir que existem infinitas soluções. Logo, o conjunto de vetores é LD.

O argumento que utilizamos no Exemplo 3.1.2 acima é geral e se aplica em várias situações. Temos

Se k>m, então os vetores ⁢v→1,v→2,…,v→k∈ℝm⁢ são necessariamente LD.\boxed{\text{Se $k>m$, ent\~{a}o os vetores }\vec{v}_{1},\vec{v}_{2},\dots,% \vec{v}_{k}\in\mathbb{R}^{m}\text{ s\~{a}o necessariamente LD.}} (3.11)

Desta forma, não existem quatro vetores LI em ℝ3\mathbb{R}^{3}. Não conseguiríamos encontrar sete vetores LI em ℝ5\mathbb{R}^{5} e assim por diante.

Façamos agora algumas observações gerais sobre independência linear:

  • •
    ​

    O vetor nulo, mesmo que sozinho, é LD. Se v→=0→\vec{v}=\vec{0} estiver em um conjunto de vetores, este conjunto será automaticamente LD.

  • •
    ​

    Ao considerarmos apenas dois vetores u→\vec{u} e v→\vec{v}, dizer que estes vetores são LD significa que eles são múltiplos um do outro e, portanto, colineares.

  • •
    ​

    Veremos mais adiante no curso que a dimensão do espaço gerado por vetores linearmente independentes é exatamente o número de vetores. Desta maneira, concluiremos, por exemplo, que o conjunto gerado por dois vetores do espaço tridimensional é um espaço linear de dimensão dois: um plano.

ER 3.1.1.

Diga se v1→\vec{v_{1}}, v2→\vec{v_{2}} e v3→\vec{v_{3}} são vetores linearmente independentes ou dependentes.

v1→=[246],v2→=[81012],v3→=[420]\vec{v_{1}}=\left[\begin{array}[]{c}2\\ 4\\ 6\end{array}\right]\quad,\quad\vec{v_{2}}=\left[\begin{array}[]{c}8\\ 10\\ 12\end{array}\right]\quad,\quad\vec{v_{3}}=\left[\begin{array}[]{c}4\\ 2\\ 0\end{array}\right] (3.12)
Solução.

Para determinar se o conjunto de vetores dados é linearmente dependente, precisamos saber se existe solução não-trivial na seguinte equação:

x1⁢v1→+x2⁢v2→+x3⁢v3→=0→x_{1}\vec{v_{1}}+x_{2}\vec{v_{2}}+x_{3}\vec{v_{3}}=\vec{0} (3.13)

Podemos escrever também:

x1⁢[246]+x2⁢[81012]+x3⁢[420]=[000]x_{1}\left[\begin{array}[]{c}2\\ 4\\ 6\end{array}\right]+x_{2}\left[\begin{array}[]{c}8\\ 10\\ 12\end{array}\right]+x_{3}\left[\begin{array}[]{c}4\\ 2\\ 0\end{array}\right]=\left[\begin{array}[]{c}0\\ 0\\ 0\end{array}\right] (3.14)

Dessa maneira, o problema se reduz a escalonar a matriz abaixo:

[28404102061200]\left[\begin{array}[]{cccc}2&8&4&0\\ 4&10&2&0\\ 6&12&0&0\end{array}\right] (3.15)

Escalonando-a, obtemos a matriz:

[142001100000]\left[\begin{array}[]{cccc}1&4&2&0\\ 0&1&1&0\\ 0&0&0&0\end{array}\right] (3.16)

Através dela, podemos observar que x1x_{1} e x2x_{2} são variáveis dependentes, enquanto que x3x_{3} é variável livre. Ou seja, a equação inicial possui infinitas soluções além da solução trivial. Portanto, obtemos que os vetores v1v_{1}, v2v_{2} e v3v_{3} são linearmente dependentes. ⊲\lhd ∎

Matrizes simétricas e prova do Teorema Espectral

12.3 Matrizes simétricas e prova do Teorema Espectral

Nesta seção vamos utilizar a notação ⟨u→,v→⟩\langle\vec{u},\vec{v}\rangle para o produto escalar, ao invés de u→⋅v→\vec{u}\cdot\vec{v}, para enfatizar qual é o vetor u→\vec{u} e qual é o vetor v→\vec{v} (em especial quando um deles está sendo multiplicado por uma matriz).

Observação 12.3.1.

Uma matriz quadrada AA, não necessariamente simétrica, cujos coeficientes são reais, satisfaz

⟨A⁢x→,y→⟩=⟨x→,AT⁢y→⟩, para todos ⁢x→,y→∈ℝn.\langle A\vec{x},\vec{y}\rangle=\langle\vec{x},A^{T}\vec{y}\rangle,\quad\text{% para todos }\vec{x},\vec{y}\in\mathbb{R}^{n}. (12.78)

Isto se pode verificar ao analisar os coeficientes de A⁢x→A\vec{x} e de AT⁢y→A^{T}\vec{y}. Assim, uma matriz AA de ordem n×nn\times n é simétrica se, e somente se, satisfaz

⟨A⁢x→,y→⟩=⟨x→,A⁢y→⟩, para todos ⁢x→,y→∈ℝn.\langle A\vec{x},\vec{y}\rangle=\langle\vec{x},A\vec{y}\rangle,\quad\text{ % para todos }\vec{x},\vec{y}\in\mathbb{R}^{n}. (12.79)

Outras formas de escrever esta mesma coisa (concorda?) são

A⁢x→⋅y→=x→⋅A⁢y→, para todos ⁢x→,y→∈ℝnA\vec{x}\cdot\vec{y}=\vec{x}\cdot A\vec{y},\quad\text{ para todos }\vec{x},% \vec{y}\in\mathbb{R}^{n} (12.80)

e

y→T⁢A⁢x→=x→T⁢A⁢y→, para todos ⁢x→,y→∈ℝn.\vec{y}^{T}A\vec{x}=\vec{x}^{T}A\vec{y},\quad\text{ para todos }\vec{x},\vec{y% }\in\mathbb{R}^{n}. (12.81)
Observação 12.3.2.

A observação anterior nos permite estender a definição de matriz simétrica para transformações lineares, pois é independente do sistema de coordenadas utilizado (depende apenas de uma escolha de produto interno/escalar).

A equivalência que foi apresentada na Observação 12.3.1 é muito útil na análise de matrizes simétricas. Vamos ver como se aplica diversas vezes na demonstração do Teorema Espectral que será dividida em três proposições preliminares. Cada uma delas apresenta, em ordem, o procedimento usual que viemos utilizando para diagonalizar matrizes: análise dos autovalores, construção dos autoespaços e verificação de que é possível formar uma base de autovetores. Neste caso de uma matriz simétrica, veremos que, além de tudo, é possível encontrar uma base ortonormal de autovetores.

Proposição 12.3.1.

Os autovalores de uma matriz simétrica AA, de entradas reais, são números reais.

Demonstração.

Seja λ\lambda uma raiz do polinômio característico de AA, que pode ser um número real ou um número complexo. Podemos encontrar um vetor v→\vec{v}, de entradas possivelmente complexas, tal que A⁢v→=λ⁢v→A\vec{v}=\lambda\vec{v}. A menos de um passo de normalização, podemos pensar que ‖v→‖=1\|\vec{v}\|=1. Vamos verificar que de fato λ\lambda é um número real. Para isto, vamos mostrar que λ\lambda é igual ao seu conjugado, λ=λ¯\lambda=\bar{\lambda}, de forma que a parte imaginária de λ\lambda deve ser igual a zero. Sendo AA simétrica, temos que

λ¯=λ⁢‖v→‖2¯=⟨λ⁢v→,v→⟩¯=⟨A⁢v→,v→⟩¯=⟨v→,A⁢v→⟩¯=⟨v→,λ⁢v→⟩¯=λ⁢⟨v→,v→⟩¯=λ.\overline{\lambda}=\overline{\lambda\|\vec{v}\|^{2}}=\overline{\langle\lambda% \vec{v},\vec{v}\rangle}=\overline{\langle A\vec{v},\vec{v}\rangle}=\overline{% \langle\vec{v},A\vec{v}\rangle}=\overline{\langle\vec{v},\lambda\vec{v}\rangle% }=\lambda\overline{\langle\vec{v},\vec{v}\rangle}=\lambda. (12.82)

A simetria de AA foi necessária (apenas) no quarto sinal de igualdade acima. Ainda, na penúltima igualdade, utilizamos sem muitos comentários uma propriedade do produto interno em espaços vetoriais com escalares complexos, a saber, que

⟨u→,v→⟩=def∑i=1nui⁢v¯i.\langle\vec{u},\vec{v}\rangle\stackrel{{\scriptstyle\text{def}}}{{=}}\sum_{i=1% }^{n}u_{i}\overline{v}_{i}. (12.83)

Os conjugados das componentes do segundo vetor aparecem para que a fórmula usual para o comprimento continue sendo válida:

‖u→‖2=⟨u→,u→⟩=∑i=1nui⁢u¯i=∑i=1n|ui|2.\|\vec{u}\|^{2}=\langle\vec{u},\vec{u}\rangle=\sum_{i=1}^{n}u_{i}\overline{u}_% {i}=\sum_{i=1}^{n}|u_{i}|^{2}. (12.84)

Desta forma, continua valendo que

⟨k⁢u→,v→⟩=k⁢⟨u→,v→⟩,enquanto que⟨u→,k⁢v→⟩=k¯⁢⟨u→,v→⟩.∎\langle k\vec{u},\vec{v}\rangle=k\langle\vec{u},\vec{v}\rangle,\quad\text{% enquanto que}\quad\langle\vec{u},k\vec{v}\rangle=\overline{k}\langle\vec{u},% \vec{v}\rangle.\qed (12.85)
Proposição 12.3.2.

Se AA é uma matriz simétrica, então autovetores associados a autovalores distintos são ortogonais.

Demonstração.

Sejam λ1\lambda_{1} e λ2\lambda_{2} autovalores distintos de AA e v→1,v→2\vec{v}_{1},\vec{v}_{2} respectivos autovetores. Então (observe que é necessário que AA seja simétrica na segunda e na última igualdade!)

λ1⁢⟨v→1,v→2⟩=⟨λ1⁢v→1,v→2⟩=⟨A⁢v→1,v→2⟩=⟨v→1,A⁢v→2⟩=λ¯2⁢⟨v→1,v→2⟩=λ2⁢⟨v→1,v→2⟩,\lambda_{1}\langle\vec{v}_{1},\vec{v}_{2}\rangle=\langle\lambda_{1}\vec{v}_{1}% ,\vec{v}_{2}\rangle=\langle A\vec{v}_{1},\vec{v}_{2}\rangle=\langle\vec{v}_{1}% ,A\vec{v}_{2}\rangle=\overline{\lambda}_{2}\langle\vec{v}_{1},\vec{v}_{2}% \rangle=\lambda_{2}\langle\vec{v}_{1},\vec{v}_{2}\rangle, (12.86)

de modo que (λ1−λ2)⁢⟨v→1,v→2⟩=0(\lambda_{1}-\lambda_{2})\langle\vec{v}_{1},\vec{v}_{2}\rangle=0. Já que estamos supondo que os autovalores são distintos, λ1≠λ2\lambda_{1}\neq\lambda_{2}, concluímos que ⟨v→1,v→2⟩=0\langle\vec{v}_{1},\vec{v}_{2}\rangle=0. ∎

Proposição 12.3.3.

Seja AA uma matriz simétrica. Então, a dimensão do autoespaço associado a um autovalor é igual à multiplicidade deste autovalor.

Demonstração.

Suponhamos que λ1\lambda_{1} é um autovalor de AA de multiplicidade kk. Seja v→1\vec{v}_{1} um autovetor unitário associado. Podemos encontrar uma base ortonormal de ℝn\mathbb{R}^{n} da qual v→1\vec{v}_{1} faz parte, isto é, podemos encontrar vetores unitários y→j\vec{y}_{j} de modo que

{v→1,y→2,y→3,…,y→n}\{\vec{v}_{1},\vec{y}_{2},\vec{y}_{3},\dots,\vec{y}_{n}\} (12.87)

é uma base ortonormal de ℝn\mathbb{R}^{n}.

Considere a matriz QQ cujas colunas são os elementos desta base:

Q=[||||v→1y→2y→3⋯y→n||||],Q=\begin{bmatrix}|&|&|&&|\\ \vec{v}_{1}&\vec{y}_{2}&\vec{y}_{3}&\cdots&\vec{y}_{n}\\ |&|&|&&|\\ \end{bmatrix}, (12.88)

que é, consequentemente, ortogonal. Vamos calcular Q−1⁢A⁢Q=QT⁢A⁢QQ^{-1}AQ=Q^{T}AQ. O produto A⁢QAQ pode ser interpretado como o produto de AA por cada uma das colunas de QQ:

A⁢Q=[||||A⁢v→1A⁢y→2A⁢y→3⋯A⁢y→n||||]=[||||λ1⁢v→1A⁢y→2A⁢y→3⋯A⁢y→n||||].AQ=\begin{bmatrix}|&|&|&&|\\ A\vec{v}_{1}&A\vec{y}_{2}&A\vec{y}_{3}&\cdots&A\vec{y}_{n}\\ |&|&|&&|\\ \end{bmatrix}=\begin{bmatrix}|&|&|&&|\\ \lambda_{1}\vec{v}_{1}&A\vec{y}_{2}&A\vec{y}_{3}&\cdots&A\vec{y}_{n}\\ |&|&|&&|\\ \end{bmatrix}. (12.89)

Logo,

QT⁢A⁢Q=[—v→1——y→2——y→3—⋮—y→n—]⁢[||||λ1⁢v→1A⁢y→2A⁢y→3⋯A⁢y→n||||]Q^{T}AQ=\begin{bmatrix}\text{---}&\vec{v}_{1}&\text{---}\\ \text{---}&\vec{y}_{2}&\text{---}\\ \text{---}&\vec{y}_{3}&\text{---}\\ &\vdots&\\ \text{---}&\vec{y}_{n}&\text{---}\\ \end{bmatrix}\begin{bmatrix}|&|&|&&|\\ \lambda_{1}\vec{v}_{1}&A\vec{y}_{2}&A\vec{y}_{3}&\cdots&A\vec{y}_{n}\\ |&|&|&&|\\ \end{bmatrix} (12.90)
⟹QT⁢A⁢Q=[λ1⁢⟨v→1,v→1⟩⟨v→1,A⁢y→2⟩⟨v→1,A⁢y→3⟩⋯⟨v→1,A⁢y→n⟩λ1⁢⟨y→2,v→1⟩⟨y→2,A⁢y→2⟩⟨y→2,A⁢y→3⟩⋯⟨y→2,A⁢y→n⟩λ1⁢⟨y→3,v→1⟩⟨y→3,A⁢y→2⟩⟨y→3,A⁢y→3⟩⋯⟨y→3,A⁢y→n⟩⋮⋮⋮⋱⋮λ1⁢⟨y→n,v→1⟩⟨y→n,A⁢y→2⟩⟨y→n,A⁢y→3⟩⋯⟨y→n,A⁢y→n⟩]\implies Q^{T}AQ=\begin{bmatrix}\lambda_{1}\langle\vec{v}_{1},\vec{v}_{1}% \rangle&\langle\vec{v}_{1},A\vec{y}_{2}\rangle&\langle\vec{v}_{1},A\vec{y}_{3}% \rangle&\cdots&\langle\vec{v}_{1},A\vec{y}_{n}\rangle\\ \lambda_{1}\langle\vec{y}_{2},\vec{v}_{1}\rangle&\langle\vec{y}_{2},A\vec{y}_{% 2}\rangle&\langle\vec{y}_{2},A\vec{y}_{3}\rangle&\cdots&\langle\vec{y}_{2},A% \vec{y}_{n}\rangle\\ \lambda_{1}\langle\vec{y}_{3},\vec{v}_{1}\rangle&\langle\vec{y}_{3},A\vec{y}_{% 2}\rangle&\langle\vec{y}_{3},A\vec{y}_{3}\rangle&\cdots&\langle\vec{y}_{3},A% \vec{y}_{n}\rangle\\ \vdots&\vdots&\vdots&\ddots&\vdots\\ \lambda_{1}\langle\vec{y}_{n},\vec{v}_{1}\rangle&\langle\vec{y}_{n},A\vec{y}_{% 2}\rangle&\langle\vec{y}_{n},A\vec{y}_{3}\rangle&\cdots&\langle\vec{y}_{n},A% \vec{y}_{n}\rangle\\ \end{bmatrix} (12.91)

Observamos que, pela simetria de AA e por ser {v→1,y→2,y→3,…,y→n}\{\vec{v}_{1},\vec{y}_{2},\vec{y}_{3},\dots,\vec{y}_{n}\} um conjunto ortonormal, temos

⟨v→1,A⁢y→j⟩=⟨A⁢v→1,y→j⟩=λ1⁢⟨v→1,y→j⟩=0,para qualquer índice ⁢j.\langle\vec{v}_{1},A\vec{y}_{j}\rangle=\langle A\vec{v}_{1},\vec{y}_{j}\rangle% =\lambda_{1}\langle\vec{v}_{1},\vec{y}_{j}\rangle=0,\quad\text{para qualquer % \'{\i}ndice }j. (12.92)

Logo,

⟹QT⁢A⁢Q=[λ100⋯00b22b23⋯b2⁢n0b32b33⋯b3⁢n⋮⋮⋮⋱⋮0bn⁢2bn⁢3⋯bn⁢n]=defA^.\implies Q^{T}AQ=\begin{bmatrix}\lambda_{1}&0&0&\cdots&0\\ 0&b_{22}&b_{23}&\cdots&b_{2n}\\ 0&b_{32}&b_{33}&\cdots&b_{3n}\\ \vdots&\vdots&\vdots&\ddots&\vdots\\ 0&b_{n2}&b_{n3}&\cdots&b_{nn}\\ \end{bmatrix}\stackrel{{\scriptstyle\text{def}}}{{=}}\hat{A}. (12.93)

Chamamos os coeficientes restantes de bi⁢jb_{ij} simplesmente porque não nos interessa muito quanto valem (notamos, no entanto, que a simetria de AA implica que bi⁢j=bj⁢ib_{ij}=b_{ji}). Observamos que

det(A−λ⁢I)=det(Q−1⁢(A−λ⁢I)⁢Q)=det(A^−λ⁢I)=(λ1−λ)⋅det(B−λ⁢I),\det\left(A-\lambda I\right)=\det\left(Q^{-1}(A-\lambda I)Q\right)=\det\left(% \hat{A}-\lambda I\right)=(\lambda_{1}-\lambda)\cdot\det(B-\lambda I), (12.94)

onde BB é a matriz de ordem (n−1)×(n−1)(n-1)\times(n-1) dada por (bi⁢j)(b_{ij}). Além disso, se a multiplicidade de λ1\lambda_{1} como autovalor de AA for maior do que 11, devemos ter, pela igualdade acima, que λ1\lambda_{1} também é um autovalor de BB. Desta forma, A^\hat{A} possui um autovetor unitário v→2\vec{v}_{2} associado com λ1\lambda_{1} que está contido em Span⁡{y→2,y→3,…,y→n}\operatorname{Span}\{\vec{y}_{2},\vec{y}_{3},\dots,\vec{y}_{n}\} e, portanto, é distinto e linearmente independente a v→1\vec{v}_{1}. O processo de Gram-Schmidt permite obter {v→1,v→2}\{\vec{v}_{1},\vec{v}_{2}\} ortonormais.

A partir daí, repetimos o processo completando o conjunto {v→1,v→2}\{\vec{v}_{1},\vec{v}_{2}\} a uma base ortonormal {v→1,v→2,y→3,…,y→n}\{\vec{v}_{1},\vec{v}_{2},\vec{y}_{3},\dots,\vec{y}_{n}\} de ℝn\mathbb{R}^{n}. Daí vamos obter um terceiro autovetor ortogonal aos anteriores e unitário. Procedemos desta maneira até encontrar kk autovetores ortogonais e unitários associados ao autovalor λ1\lambda_{1}. Assim, o autoespaço Nul⁡(A−λ1⁢I)\operatorname{Nul}(A-\lambda_{1}I) tem dimensão kk, que é igual à multiplicidade do autovalor λ1\lambda_{1}. ∎

12.3.1 Prova do Teorema Espectral

O Teorema Fundamental da Álgebra afirma que todo polinômio não constante de grau nn possui exatamente nn raízes complexas. Sendo assim, qualquer matriz AA de ordem n×nn\times n possui nn autovalores complexos. No entanto, sendo AA uma matriz simétrica, a Proposição 12.3.1 garante que todas estes nn autovalores são todos reais. Vamos denotar por

λ1,λ2,…,λk∈ℝ\lambda_{1},\lambda_{2},\dots,\lambda_{k}\in\mathbb{R} (12.95)

os autovalores distintos de AA e por m1,m2,…,mkm_{1},m_{2},\dots,m_{k} suas respectivas multiplicidades. Temos, em particular, que m1+m2+⋯+mk=nm_{1}+m_{2}+\cdots+m_{k}=n.

Pela Proposição 12.3.3, a dimensão do autoespaço associado a λ1\lambda_{1} é m1m_{1}. Podemos encontrar (da maneira usual, por escalonamento) uma base de autovetores

Nul⁡(A−λ1⁢I)=Span⁡{u→1,u→2,…,u→m1}.\operatorname{Nul}(A-\lambda_{1}I)=\operatorname{Span}\{\vec{u}_{1},\vec{u}_{2% },\dots,\vec{u}_{m_{1}}\}. (12.96)

Estes autovetores, possivelmente não são ortogonais, mas, aplicando o Processo de Gram-Schmidt, podemo obter uma base ortonormal de Nul⁡(A−λ1⁢I)\operatorname{Nul}(A-\lambda_{1}I):

Nul⁡(A−λ1⁢I)=Span⁡{v→1,v→2,…,v→m1}.\operatorname{Nul}(A-\lambda_{1}I)=\operatorname{Span}\{\vec{v}_{1},\vec{v}_{2% },\dots,\vec{v}_{m_{1}}\}. (12.97)

Fazendo este mesmo procedimento para cada um dos autovalores, obtemos nn autovetores, que, consequentemente, formam uma base para ℝn\mathbb{R}^{n}. Logo, AA é diagonalizável.

Além disso, a Proposição 12.3.2 garante que todo autovetor de um autoespaço é ortogonal a todo autovetor de outro autoespaço (associado a um autovalor diferente). Temos então kk autoespaços, cujas bases são ortonormais e elementos da base de um autoespaço são ortogonais aos elementos das outras bases. Segue daí que a união das bases dos autoespaços, como consideramos, é uma base ortonormal para ℝn\mathbb{R}^{n}.

Explicitamente, a matriz

P=[|||v→1v→2⋯v→n|||]P=\begin{bmatrix}|&|&&|\\ \vec{v}_{1}&\vec{v}_{2}&\cdots&\vec{v}_{n}\\ |&|&&|\\ \end{bmatrix} (12.98)

formada pelos autovetores é ortogonal e

P−1⁢A⁢P=D=[λ100⋯00λ20⋯000λ3⋯0⋮⋮⋮⋱⋮000⋯λn].P^{-1}AP=D=\begin{bmatrix}\lambda_{1}&0&0&\cdots&0\\ 0&\lambda_{2}&0&\cdots&0\\ 0&0&\lambda_{3}&\cdots&0\\ \vdots&\vdots&\vdots&\ddots&\vdots\\ 0&0&0&\cdots&\lambda_{n}\\ \end{bmatrix}. (12.99)

Sendo PP ortogonal, tem-se ainda que P−1=PTP^{-1}=P^{T}.

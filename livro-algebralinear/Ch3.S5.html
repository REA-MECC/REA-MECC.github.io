Transformações injetoras, sobrejetoras e invertíveis

3.5 Transformações injetoras, sobrejetoras e invertíveis

Como de costume, dada uma função f:A→Bf:A\to B, diz-se que AA é o domínio de ff enquanto BB é o contradomínio. A imagem de ff é o subconjunto de BB que consiste de todos os elementos y∈By\in B tais que f⁢(x)=yf(x)=y ou, intuitivamente, que consiste de “todos os elementos de BB que são atingidos pela função ff”.

Na hora de decidir se uma função é invertível ou não, duas propriedades são essenciais:

  1. 1)1)
    ​

    cada elemento de BB ser a imagem de no máximo um elemento de AA, caso em que ff é dita injetora ou injetiva;

  2. 2)2)
    ​

    a imagem de ff ser igual ao contradomínio, caso em que ff diz-se sobrejetora ou sobrejetiva.

Quando estas duas propriedades são satisfeitas, isto é, quando a função ff é injetora e sobrejetora, vale que ff é invertível: podemos encontrar uma função f−1:B→Af^{-1}:B\to A que satisfaz

f−1⁢(f⁢(x))=x,para todo ⁢x∈Aef⁢(f−1⁢(y))=y,para todo ⁢y∈B.f^{-1}\big{(}f(x)\big{)}=x,\ \text{para todo }x\in A\quad\text{e}\quad f\big{(% }f^{-1}(y)\big{)}=y,\ \text{para todo }y\in B. (3.44)

Ao tentarmos encontrar uma função inversa f−1f^{-1}, as propriedades de ser injetora e sobrejetora, aparecem naturalmente.

Estas definições são usuais para funções quaisquer. A partir de agora, vamos analisar quando que transformações lineares são injetoras, sobrejetoras e/ou invertíveis. Vamos ver, em particular, que é bem mais fácil estudar tais propriedades para transformações lineares.

3.5.1 Transformações lineares injetoras

A transformação linear T:ℝn→ℝmT:\mathbb{R}^{n}\to\mathbb{R}^{m} é injetora quando, para b→∈ℝm\vec{b}\in\mathbb{R}^{m}, a equação

T⁢(x→)=b→T(\vec{x})=\vec{b} (3.45)

possuir uma única solução ou nenhuma (no máximo uma – comparar com a definição acima). Como vimos, existe uma matriz AA de ordem m×nm\times n associada à transformação linear TT, de modo que temos que analisar as soluções de

A⁢x→=b→.A\vec{x}=\vec{b}. (3.46)

Recaímos novamente em um sistema linear!

No caso particular em que b→=0→\vec{b}=\vec{0}, o sistema homogêneo A⁢x→=0→A\vec{x}=\vec{0} sempre possui a solução trivial x→=0→\vec{x}=\vec{0}. Neste caso, para que a transformação linear seja injetora devemos verificar que esta é a única solução de A⁢x→=0→A\vec{x}=\vec{0}. Na verdade, é possível verificar o seguinte.

Proposição 3.5.1.

TT é injetora se, e somente se, A⁢x→=0→A\vec{x}=\vec{0} possui apenas a solução trivial.22 2 Em particular, este caso somente é possível se n≤mn\leq m (reler observações que seguem o Exemplo 3)

Justificativa da afirmação.

Vamos provar matematicamente que, se soubermos que A⁢x→=0→A\vec{x}=\vec{0} possui apenas a solução trivial, então vale que A⁢x→=b→A\vec{x}=\vec{b} possui no máximo uma solução, para qualquer escolha de vetor b→\vec{b}. Isto, por sua vez, implica que TT é injetora. Caso não existam soluções ou caso exista apenas uma, nada há para provar. Suponhamos que então que existem infinitas soluções (esta seria a única outra possibilidade). Em particular, existiriam duas distintas:

x→1≠x→2⁢ tais que ⁢A⁢x→1=b→⁢ e também ⁢A⁢x→2=b→.\vec{x}_{1}\neq\vec{x}_{2}\text{ tais que }A\vec{x}_{1}=\vec{b}\text{ e tamb\'% {e}m }A\vec{x}_{2}=\vec{b}. (3.47)

Por linearidade, temos então que A⁢(x→1−x→2)=b→−b→=0→A(\vec{x}_{1}-\vec{x}_{2})=\vec{b}-\vec{b}=\vec{0}. No entanto, nossa hipótese garante que somente existe a solução trivial para a equação A⁢x→=0→A\vec{x}=\vec{0}, de modo que deveríamos ter x→1=x→2\vec{x}_{1}=\vec{x}_{2}. Em outras palavras, se A⁢x→=0→A\vec{x}=\vec{0} possuir apenas a solução trivial, então não existe mais do que uma solução para A⁢x→=b→A\vec{x}=\vec{b}. Portanto, TT é injetora. ∎

Notem que a afirmação acima é também equivalente a seguinte afirmação.

Proposição 3.5.2.

TT é injetora se, e somente se, as colunas de AA são linearmente independentes.

Vamos deixar exemplos de injetividade para a subseção seguinte (ver Exemplos 3.5.1 e 3.5.2).

3.5.2 Transformações lineares sobrejetoras

A transformação linear T:ℝn→ℝmT:\mathbb{R}^{n}\to\mathbb{R}^{m} é sobrejetora quando, para todo b→∈ℝn\vec{b}\in\mathbb{R}^{n}, a equação

T⁢(x→)=b→T(\vec{x})=\vec{b} (3.48)

possui alguma solução (comparar com a definição no início desta seção). Seja AA a matriz de ordem m×nm\times n associada a TT. Assim, para verificar que TT é sobrejetora, devemos verificar que, para qualquer b→∈ℝm\vec{b}\in\mathbb{R}^{m}, o sistema linear

A⁢x→=b→A\vec{x}=\vec{b} (3.49)

possui ao menos uma solução (uma ou infinitas). Isto é equivalente a:

T é sobrejetora se, e somente se, as colunas de A geram todo o espaço ℝm.\boxed{\text{$T$ \'{e} sobrejetora se, e somente se, as colunas de $A$ geram % todo o espa\c{c}o $\mathbb{R}^{m}$.}} (3.50)

Em particular, este caso somente é possível se n≥mn\geq m (veremos mais adiante no curso – a imagem da transformação T:ℝn→ℝmT:\mathbb{R}^{n}\to\mathbb{R}^{m} será de dimensão no máximo igual a nn).

Exemplo 3.5.1.

Considere a transformação linear TT cuja matriz associada é

A=[53110−11−10003].A=\left[\begin{array}[]{rrrr}5&3&1&1\\ 0&-1&1&-1\\ 0&0&0&3\\ \end{array}\right]. (3.51)

Como são quatro colunas de ℝ3\mathbb{R}^{3}, vemos que as colunas são LD e, portanto, TT não é injetora.

Por outro lado, a matriz já está em forma escalonada. Vamos analisar se o sistema

A⁢x→=b→⇔[5311b10−11−1b20003b3]A\vec{x}=\vec{b}\iff\left[\begin{array}[]{rrrr|r}5&3&1&1&b_{1}\\ 0&-1&1&-1&b_{2}\\ 0&0&0&3&b_{3}\\ \end{array}\right] (3.52)

possui solução para todo b→∈ℝ3\vec{b}\in\mathbb{R}^{3}. De fato, o sistema possui solução (já que nenhuma linha da sua forma escalonada é inconsistente). Em verdade, o sistema possui uma variável livre. Logo, TT é sobrejetora.

Exemplo 3.5.2.

Considere a transformação linear TT cuja matriz associada é

A=[31570−4].A=\left[\begin{array}[]{rrrr}3&1\\ 5&7\\ 0&-4\\ \end{array}\right]. (3.53)

Como são somente duas colunas, é fácil ver que uma não é múltipla da outra: por causa das primeiras componentes, podemos pensar que a primeira coluna é três vezes a primeira, mas verificando a segunda linha já não dá certo 3⋅7≠53\cdot 7\neq 5. Logo, as colunas são LI e a transformação TT é injetora.

Por outro lado, as colunas de AA geram um espaço de dimensão no máximo 2; logo, não geram ℝ3\mathbb{R}^{3}. Portanto, TT não é sobrejetora.

3.5.3 Transformações lineares invertíveis

Pelas observações das últimas subseções, só é possível da transformação linear T:ℝn→ℝmT:\mathbb{R}^{n}\to\mathbb{R}^{m} ser invertível quando m=nm=n, pois TT deverá ser ao mesmo tempo injetora e sobrejetora. Veremos na semana seguinte um método para calcular a transformação inversa T−1T^{-1}, que será também uma transformação linear.

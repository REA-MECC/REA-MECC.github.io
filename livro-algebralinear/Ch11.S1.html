Método dos mínimos quadrados

11.1 Método dos mínimos quadrados

Sistemas lineares aparecem como modelos matemáticos de vários fenômenos e em várias situações. Acontece que alguns sistemas simplesmente não possuem soluções e ficamos sem saber como proceder. O método dos mínimos quadrados é uma técnica que nos permite, de forma aproximada, retirar alguma informação desses sistemas impossíveis. A terminologia se deve ao fato de que, como veremos, este método minimiza a soma dos quadrados dos erros obtidos na aproximação.

Como sabemos, resolver o sistema linear

A⁢x→=b→A\vec{x}=\vec{b} (11.1)

consiste em encontrar um vetor x→\vec{x} que satisfaça a esta equação. Na terminologia que construimos ao longo do curso, é equivalente dizer que b→\vec{b} pertence ao espaço coluna da matriz AA, isto é, b→∈Col⁡A\vec{b}\in\operatorname{Col}A. Desta forma, não é possível resolver o sistema quando b→∉Col⁡A\vec{b}\not\in\operatorname{Col}A. Representamos esta situação na figura abaixo.

O método dos mínimos quadrados consiste em olhar para o vetor p→=projCol⁡A⁡b→\vec{p}=\operatorname{proj}_{\operatorname{Col}A}\vec{b} e resolver o sistema linear associado

A⁢x→=p→.A\vec{x}=\vec{p}. (11.2)

Esta solução de A⁢x→=p→A\vec{x}=\vec{p} é chamada de solução de mínimos quadrados. A ideia é (ver figura) considerar o vetor pertencente a Col⁡A\operatorname{Col}A que é o “mais próximo” possível de b→\vec{b} e cujo sistema linear associado possua solução. Neste contexto, estamos pensando em mais próximo no sentido

‖b→−projCol⁡A⁡b→‖≤‖b→−c→‖, para qualquer ⁢c→∈Col⁡A,\|\vec{b}-\operatorname{proj}_{\operatorname{Col}A}\vec{b}\|\leq\|\vec{b}-\vec% {c}\|,\text{ para qualquer }\vec{c}\,\in\operatorname{Col}A, (11.3)

isto é, no sentido de ser a projeção a melhor aproximação de b→\vec{b} no espaço coluna de AA. Escrevendo os vetores em coordenadas:

b→=[b1b2⋮bn]eprojCol⁡A⁡b→=[p1p2⋮pn],\vec{b}=\begin{bmatrix}b_{1}\\ b_{2}\\ \vdots\\ b_{n}\end{bmatrix}\quad\text{e}\quad\operatorname{proj}_{\operatorname{Col}A}% \vec{b}=\begin{bmatrix}p_{1}\\ p_{2}\\ \vdots\\ p_{n}\end{bmatrix}, (11.4)

podemos definir o valor

‖b→−projCol⁡A⁡b→‖=∑i=1n(bi−pi)2\|\vec{b}-\operatorname{proj}_{\operatorname{Col}A}\vec{b}\|=\sqrt{\sum_{i=1}^% {n}(b_{i}-p_{i})^{2}} (11.5)

como o erro da aproximação ou erro quadrático. Logo, como anunciado, a soma dos quadrados dos erros obtidos cada componente é o mínimo possível.

Exemplo 11.1.1.

Considere o sistema linear

{x1+x2−2⁢x3=3x1−2⁢x3=2x2+2⁢x3=0x1+x2−2⁢x3=0↭A⁢x→=[11−210−2012−1−12]⁢[x1x2x3]=[3200]=b→.\left\{\begin{array}[]{ll}x_{1}+x_{2}-2x_{3}=3\\ x_{1}-2x_{3}=2\\ x_{2}+2x_{3}=0\\ x_{1}+x_{2}-2x_{3}=0\end{array}\right.\ \leftrightsquigarrow\ A\vec{x}=\begin{% bmatrix}1&1&-2\\ 1&0&-2\\ 0&1&2\\ -1&-1&2\end{bmatrix}\begin{bmatrix}x_{1}\\ x_{2}\\ x_{3}\end{bmatrix}=\begin{bmatrix}3\\ 2\\ 0\\ 0\end{bmatrix}=\vec{b}. (11.6)

Este sistema é claramente impossível, pois a primeira equação é inconsistente com a última. De forma mais automática, um passo no escalonamento da matriz aumentada associada revelaria a mesma conclusão:

[A|b→]=[11−2310−220120−1−120]∼[11−2310−2201200003].[\,A\ |\ \vec{b}\,]=\begin{bmatrix}1&1&-2&3\\ 1&0&-2&2\\ 0&1&2&0\\ -1&-1&2&0\end{bmatrix}\sim\begin{bmatrix}1&1&-2&3\\ 1&0&-2&2\\ 0&1&2&0\\ 0&0&0&3\end{bmatrix}. (11.7)

Vamos tentar encontrar, conforme descrito acima, uma solução de mínimos quadrados. Passos a serem realizados:

  • •
    ​

    Para calcular a projeção de b→\vec{b} sobre Col⁡A\operatorname{Col}A, precisamos, em primeiro lugar, obter uma base ortogonal do espaço Col⁡A\operatorname{Col}A; isto pode ser feito pelo processo de Gram–Schmidt;

  • •
    ​

    Calcular a projeção p→=projCol⁡A⁡b→\vec{p}=\operatorname{proj}_{\operatorname{Col}A}\vec{b};

  • •
    ​

    Resolver o sistema linear A⁢x→=p→A\vec{x}=\vec{p}.

Embora seja possível realizar estes três passos, temos de convir que seria muito trabalhoso. Por isto, vamos analisar um pouco mais da teoria para encontrar um método mais eficaz.⊲.\ \lhd

Suponhamos que:

  • •
    ​

    AA é uma matriz m×nm\times n,

  • •
    ​

    x→∈ℝn\vec{x}\in\mathbb{R}^{n},

  • •
    ​

    b→∈ℝm\vec{b}\in\mathbb{R}^{m} e

  • •
    ​

    o sistema A⁢x→=b→A\vec{x}=\vec{b} não possui solução.

A solução de mínimos quadrados é a solução de A⁢x→=p→A\vec{x}=\vec{p}, onde p→=projCol⁡A⁡b→\vec{p}=\operatorname{proj}_{\operatorname{Col}A}\vec{b}. Observamos, por outro lado, que, sendo p→\vec{p}\, a projeção sobre o espaço coluna de AA, o vetor b→−p→\vec{b}-\vec{p}\, deve ser ortogonal a todos os elementos de Col⁡A\operatorname{Col}A. Em particular, se escrevermos AA em termos de suas colunas

A=[|||a→1a→2⋯a→n|||],(que, em particular, implica ⁢AT=[—a→1——a→2—⋮—a→n—])A=\begin{bmatrix}|&|&&|\\ \vec{a}_{1}&\vec{a}_{2}&\cdots&\vec{a}_{n}\\ |&|&&|\end{bmatrix},\qquad\left(\text{que, em particular, implica }A^{T}=% \begin{bmatrix}\ \text{---}&\vec{a}_{1}&\text{---}\ \\ \ \text{---}&\vec{a}_{2}&\text{---}\ \\ \ &\vdots&\ \\ \ \text{---}&\vec{a}_{n}&\text{---}\ \end{bmatrix}\right) (11.8)

devemos ter

a→j⋅(b→−p→)=0, para todo ⁢j⇔AT⁢(b→−p→)=0→⇔AT⁢b→=AT⁢p→=AT⁢A⁢x→.\vec{a}_{j}\cdot\big{(}\vec{b}-\vec{p}\big{)}=0,\text{ para todo }j\ \ \iff\ % \ A^{T}(\vec{b}-\vec{p})=\vec{0}\ \ \iff\ \ A^{T}\vec{b}=A^{T}\vec{p}=A^{T}A% \vec{x}. (11.9)

Esta última linha é válida porque o produto escalar a→j⋅(b→−p→)\vec{a}_{j}\cdot\big{(}\vec{b}-\vec{p}\big{)} é justamente a entrada jj do vetor obtido ao multiplicar AT⁢(b→−p→)A^{T}(\vec{b}-\vec{p}). Além disso, A⁢x→=p→.A\vec{x}=\vec{p}.

Concluimos, desta forma, que, se x→\vec{x} é uma solução de mínimos quadrados, ou seja, se A⁢x→=p→A\vec{x}=\vec{p}, então necessariamente

ATAx→=ATb→.\boxed{A^{T}A\vec{x}=A^{T}\vec{b}.} (11.10)

Tudo isto implica que podemos utilizar a equação (11.10) para encontrar as soluções de mínimos quadrados de A⁢x→=b→A\vec{x}=\vec{b}.

Exemplo 11.1.2 (De volta ao Exemplo 11.1.1).

Calculamos

AT⁢A=[110−1101−1−2−222]⁢[11−210−2012−1−12]=[32−623−2−6−216]A^{T}A=\begin{bmatrix}1&1&0&-1\\ 1&0&1&-1\\ -2&-2&2&2\end{bmatrix}\begin{bmatrix}1&1&-2\\ 1&0&-2\\ 0&1&2\\ -1&-1&2\end{bmatrix}=\begin{bmatrix}3&2&-6\\ 2&3&-2\\ -6&-2&16\end{bmatrix} (11.11)

e

AT⁢b→=[110−1101−1−2−222]⁢[3200]=[53−10]A^{T}\vec{b}=\begin{bmatrix}1&1&0&-1\\ 1&0&1&-1\\ -2&-2&2&2\end{bmatrix}\begin{bmatrix}3\\ 2\\ 0\\ 0\end{bmatrix}=\begin{bmatrix}5\\ 3\\ -10\end{bmatrix} (11.12)

Para resolvermos o sistema AT⁢A⁢x→=AT⁢b→A^{T}A\vec{x}=A^{T}\vec{b}, vamos escalonar a matriz aumentada associada:

[32−6523−23−6−216−10]∼[32−65056−10240]∼[32−65056−10041]\begin{bmatrix}3&2&-6&5\\ 2&3&-2&3\\ -6&-2&16&-10\end{bmatrix}\sim\begin{bmatrix}3&2&-6&5\\ 0&5&6&-1\\ 0&2&4&0\end{bmatrix}\sim\begin{bmatrix}3&2&-6&5\\ 0&5&6&-1\\ 0&0&4&1\end{bmatrix} (11.13)

Logo,

{3⁢x1+2⁢x2−6⁢x3=55⁢x2+6⁢x3=−14⁢x3=1.\left\{\begin{array}[]{ll}3x_{1}+2x_{2}-6x_{3}=5\\ 5x_{2}+6x_{3}=-1\\ 4x_{3}=1\end{array}\right.. (11.14)

Assim,

x3=14⟹5⁢x2+6⋅14=−1⟹x2=−12⟹3⁢x1−1−32=5⟹x1=32.x_{3}=\frac{1}{4}\implies 5x_{2}+6\cdot\frac{1}{4}=-1\implies x_{2}=-\frac{1}{% 2}\implies 3x_{1}-1-\frac{3}{2}=5\implies x_{1}=\frac{3}{2}. (11.15)

Portanto, uma solução de mínimos quadrados é

[x1x2x3]=[3/2−1/21/4].\begin{bmatrix}x_{1}\\ x_{2}\\ x_{3}\end{bmatrix}=\begin{bmatrix}3/2\\ -1/2\\ 1/4\end{bmatrix}. (11.16)

Observe como isto é muito menos trabalhoso do que o método que havíamos esquematizado no Exemplo 11.1.1.⊲.\ \lhd

Observação 11.1.1.

Caso a matriz AA já seja uma matriz ortogonal, podemos calcular projeções diretamente, pois uma base ortogonal para Col⁡A\operatorname{Col}A já esté disponível desde o começo. Neste caso, ambos os métodos exigiriam um trabalho parecido.

Nossa aproximação linear foi encontrada de modo a minimizar o erro quadrático. Para encontrar o erro, deveríamos calcular

‖b→−projCol⁡A→⁡b→‖.\|\vec{b}-\operatorname{proj}_{\vec{\operatorname{Col}A}}\vec{b}\|. (11.17)

No entanto, note que não precisamos calcular a projeção. Observando que a solução de mínimos quadrados satisfaz A⁢x→=projCol⁡A→⁡b→A\vec{x}=\operatorname{proj}_{\vec{\operatorname{Col}A}}\vec{b}, podemos calcular o erro por

erro =‖b→−A⁢x→‖,\text{erro }=\|\vec{b}-A\vec{x}\|, (11.18)

onde x→\vec{x} é a solução de mínimos quadrados.

No Exemplo 11.1.2 acima, podemos calcular o erro desta forma. De fato,

A⁢x→=[11−210−2012−1−12]⁢[3/2−1/21/4]=[1/210−1/2]⟹b→−A⁢x→=[5/2101/2]A\vec{x}=\begin{bmatrix}1&1&-2\\ 1&0&-2\\ 0&1&2\\ -1&-1&2\end{bmatrix}\begin{bmatrix}3/2\\ -1/2\\ 1/4\end{bmatrix}=\begin{bmatrix}1/2\\ 1\\ 0\\ -1/2\end{bmatrix}\implies\vec{b}-A\vec{x}=\begin{bmatrix}5/2\\ 1\\ 0\\ 1/2\end{bmatrix} (11.19)
⟹erro =‖b→−A⁢x→‖=254+1+14=302≃2,74.\implies\text{erro }=\|\vec{b}-A\vec{x}\|=\sqrt{\frac{25}{4}+1+\frac{1}{4}}=% \frac{\sqrt{30}}{2}\simeq 2,74. (11.20)

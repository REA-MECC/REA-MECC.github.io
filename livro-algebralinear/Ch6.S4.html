Matriz de mudança de coordenadas

6.4 Matriz de mudança de coordenadas

Se conhecemos as coordenadas do vetor x→\vec{x} em uma base ℬ={v→1,v→2,…,v→n}\mathcal{B}=\big{\{}\vec{v}_{1},\vec{v}_{2},\dots,\vec{v}_{n}\big{\}}:

[x→]ℬ=[b1b2⋮bn]ℬ=b1⁢v→1+b2⁢v→2+⋯+bn⁢v→n,\big{[}\vec{x}\big{]}_{\mathcal{B}}=\left[\begin{array}[]{c}b_{1}\\ b_{2}\\ \vdots\\ b_{n}\\ \end{array}\right]_{\mathcal{B}}=b_{1}\vec{v}_{1}+b_{2}\vec{v}_{2}+\cdots+b_{n% }\vec{v}_{n}, (6.42)

podemos obter as coordenadas usuais (na base canônica) de x→\vec{x} de forma sistemática. Em outras palavras, queremos encontrar x1,x2,…,xnx_{1},x_{2},\dots,x_{n} tais que

x→=[x1x2⋮xn]=x1⁢e→1+x2⁢e→2+⋯+xn⁢e→n.\vec{x}=\left[\begin{array}[]{c}x_{1}\\ x_{2}\\ \vdots\\ x_{n}\\ \end{array}\right]=x_{1}\vec{e}_{1}+x_{2}\vec{e}_{2}+\cdots+x_{n}\vec{e}_{n}. (6.43)

Uma maneira é considerar a matriz cujas colunas são os vetores v→i\vec{v}_{i}:

A=[|||v→1v→2⋯v→n|||]=[v11v12⋯v1⁢nv21v22⋯v2⁢n⋮⋮⋮vn⁢1vn⁢2⋯vn⁢n].A=\left[\begin{array}[]{cccc}|&|&&|\\ \vec{v}_{1}&\vec{v}_{2}&\cdots&\vec{v}_{n}\\ |&|&&|\\ \end{array}\right]=\left[\begin{array}[]{cccc}v_{11}&v_{12}&\cdots&v_{1n}\\ v_{21}&v_{22}&\cdots&v_{2n}\\ \vdots&\vdots&&\vdots\\ v_{n1}&v_{n2}&\cdots&v_{nn}\\ \end{array}\right]. (6.44)

Funciona porque

b1⁢v→1+b2⁢v→2+⋯+bn⁢v→n\displaystyle b_{1}\vec{v}_{1}+b_{2}\vec{v}_{2}+\cdots+b_{n}\vec{v}_{n} =∑j=1nbj⁢v→j=∑j=1nbj⁢∑i=1nvi⁢j⁢e→i=∑i=1n(∑j=1nbj⁢vi⁢j)⁢e→i\displaystyle=\sum_{j=1}^{n}b_{j}\vec{v}_{j}=\sum_{j=1}^{n}b_{j}\sum_{i=1}^{n}% v_{ij}\vec{e}_{i}=\sum_{i=1}^{n}\bigg{(}\sum_{j=1}^{n}b_{j}v_{ij}\bigg{)}\vec{% e}_{i}
=(∑j=1nbj⁢v1⁢j)⁢e→1+(∑j=1nbj⁢v2⁢j)⁢e→2+⋯+(∑j=1nbj⁢vn⁢j)⁢e→n.\displaystyle=\bigg{(}\sum_{j=1}^{n}b_{j}v_{1j}\bigg{)}\vec{e}_{1}+\bigg{(}% \sum_{j=1}^{n}b_{j}v_{2j}\bigg{)}\vec{e}_{2}+\cdots+\bigg{(}\sum_{j=1}^{n}b_{j% }v_{nj}\bigg{)}\vec{e}_{n}.

Assim, devemos ter

{x1=v11⁢b1+v12⁢b2+⋯+v1⁢n⁢bnx2=v21⁢b1+v22⁢b2+⋯+v2⁢n⁢bn⋮xn=vn⁢1⁢b1+vn⁢2⁢b2+⋯+vn⁢n⁢bni.ex→=A⁢[x→]ℬ.\left\{\begin{array}[]{lcl}x_{1}&=&v_{11}b_{1}+v_{12}b_{2}+\cdots+v_{1n}b_{n}% \\ x_{2}&=&v_{21}b_{1}+v_{22}b_{2}+\cdots+v_{2n}b_{n}\\ &\vdots&\\ x_{n}&=&v_{n1}b_{1}+v_{n2}b_{2}+\cdots+v_{nn}b_{n}\\ \end{array}\right.\quad\text{i.e}\quad\vec{x}=A[\vec{x}]_{\mathcal{B}}. (6.45)

A matriz AA construída como acima é chamada de matriz de mudança de coordenadas da base ℬ\mathcal{B} para a base usual (canônica) {e→1,e→2,…,e→n}\{\vec{e}_{1},\vec{e}_{2},\dots,\vec{e}_{n}\}.

A matriz de mudança de coordenadas da base canônica para a base ℬ\mathcal{B} pode ser obtida a partir da inversa de AA:

[x→]ℬ=A−1⋅x→.[\vec{x}]_{\mathcal{B}}=A^{-1}\cdot\vec{x}. (6.46)

(Notamos que a matriz AA é necessariamente invertível, já que suas colunas são elementos de uma base; em particular, são linearmente independentes).

Exemplo 6.4.1.

Consideramos a seguinte base para ℝ2\mathbb{R}^{2}:

ℬ={[11],[−11]}.\mathcal{B}=\left\{\left[\begin{array}[]{c}1\\ 1\\ \end{array}\right],\left[\begin{array}[]{c}-1\\ 1\\ \end{array}\right]\right\}. (6.47)

Este conjunto é de fato uma base, pois são linearmente independentes e são dois (que é a dimensão de ℝ2\mathbb{R}^{2}). Sabendo que as coordenadas de u→\vec{u} e v→\vec{v} nesta base ℬ\mathcal{B} são

[u]ℬ=[11]ℬe[v]ℬ=[3−1]ℬ,[u]_{\mathcal{B}}=\left[\begin{array}[]{c}1\\ 1\\ \end{array}\right]_{\mathcal{B}}\quad\text{e}\quad[v]_{\mathcal{B}}=\left[% \begin{array}[]{c}3\\ -1\\ \end{array}\right]_{\mathcal{B}}, (6.48)

encontrar as componentes de u→\vec{u} e v→\vec{v} na base canônica {e→1,e→2}\{\vec{e}_{1},\vec{e}_{2}\} de ℝ2\mathbb{R}^{2}.

Pelo que analisamos acima, a matriz de mudança de coordenadas que procuramos é a matriz

A=[1−111]A=\left[\begin{array}[]{cc}1&-1\\ 1&1\\ \end{array}\right] (6.49)

Logo,

[1−111]⁢[11]ℬ=[02]⁢ e ⁢v→=[1−111]⁢[3−1]ℬ=[42].\left[\begin{array}[]{cc}1&-1\\ 1&1\\ \end{array}\right]\left[\begin{array}[]{c}1\\ 1\\ \end{array}\right]_{\mathcal{B}}=\left[\begin{array}[]{c}0\\ 2\\ \end{array}\right]\text{ e }\vec{v}=\left[\begin{array}[]{cc}1&-1\\ 1&1\\ \end{array}\right]\left[\begin{array}[]{c}3\\ -1\\ \end{array}\right]_{\mathcal{B}}=\left[\begin{array}[]{c}4\\ 2\\ \end{array}\right]. (6.50)
Observação 6.4.1.

A matriz de mudança de coordenadas é uma forma de trocar de uma base para outra de forma sistemática. Com o intuito de tornar mais intuitiva a construção, vamos repetir a argumentação que fizemos no início desta seção para o caso 2×22\times 2. Na base ℬ\mathcal{B} do exemplo anterior, a representação

[b1b2]ℬ⁢ significa que ⁢v→=b1⋅[11]+b2⋅[−11].\left[\begin{array}[]{c}b_{1}\\ b_{2}\\ \end{array}\right]_{\mathcal{B}}\text{ significa que }\vec{v}=b_{1}\cdot\left[% \begin{array}[]{c}1\\ 1\\ \end{array}\right]+b_{2}\cdot\left[\begin{array}[]{c}-1\\ 1\\ \end{array}\right]. (6.51)

Os vetores da base ℬ\mathcal{B} são representados na base canônica por

[11]=[10]+[01]=e→1+e→2e[−11]=[−10]+[01]=−e→1+e→2.\left[\begin{array}[]{c}1\\ 1\\ \end{array}\right]=\left[\begin{array}[]{c}1\\ 0\\ \end{array}\right]+\left[\begin{array}[]{c}0\\ 1\\ \end{array}\right]=\vec{e}_{1}+\vec{e}_{2}\quad\text{e}\quad\left[\begin{array% }[]{c}-1\\ 1\\ \end{array}\right]=\left[\begin{array}[]{c}-1\\ 0\\ \end{array}\right]+\left[\begin{array}[]{c}0\\ 1\\ \end{array}\right]=-\vec{e}_{1}+\vec{e}_{2}. (6.52)

Logo,

v→=b1⁢(e→1+e→2)+b2⁢(−e→1+e→2)=(b1−b2)⁢e→1+(b1+b2)⁢e→2,\vec{v}=b_{1}\left(\vec{e}_{1}+\vec{e}_{2}\right)+b_{2}\left(-\vec{e}_{1}+\vec% {e}_{2}\right)=(b_{1}-b_{2})\vec{e}_{1}+(b_{1}+b_{2})\vec{e}_{2}, (6.53)

de modo que as componentes de v→\vec{v} na base canônica são

{x1=b1−b2x2=b1+b2.\left\{\begin{array}[]{ll}x_{1}=b_{1}-b_{2}\\ x_{2}=b_{1}+b_{2}\end{array}\right.. (6.54)

Em notação matricial:

[x1x2]=[1−111]⁢[b1b2].\left[\begin{array}[]{c}x_{1}\\ x_{2}\\ \end{array}\right]=\left[\begin{array}[]{cc}1&-1\\ 1&1\\ \end{array}\right]\left[\begin{array}[]{c}b_{1}\\ b_{2}\\ \end{array}\right]. (6.55)
